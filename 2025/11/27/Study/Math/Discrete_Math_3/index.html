第9章 代数系统

9.1 二元运算及其性质

1. 二元运算与一元运算

  • 二元运算：设 SSS 为集合，函数 f:S×S→Sf: S \times S \to Sf:S×S→S 称为 SSS 上的二元运算。
    • 封闭性 (Closure)：对于 SSS 中任意两个元素 a,ba, ba,b，它们的运算结果必须还在 SSS 中，且结果唯一。
    • 记号：通常用符号（如 +,∗,∘+, *, \circ+,∗,∘ 等）表示。例如 a∘b=ca \circ b = ca∘b=c。
  • 一元运算：函数 f:S→Sf: S \to Sf:S→S 称为 SSS 上的一元运算（如求倒数、取负数、求补集）。

2. 二元运算的性质

设 ∘\circ∘ 和 ∗*∗ 是集合 SSS 上的二元运算。

(1) 交换律 (Commutative Law)

  • 定义：如果对于 SSS 中任意元素 x,yx, yx,y，都有 x∘y=y∘xx \circ y = y \circ xx∘y=y∘x，则称 ∘\circ∘ 满足交换律。

(2) 结合律 (Associative Law)

  • 定义：如果对于 SSS 中任意元素 x,y,zx, y, zx,y,z，都有 (x∘y)∘z=x∘(y∘z)(x \circ y) \circ z = x \circ (y \circ z)(x∘y)∘z=x∘(y∘z)，则称 ∘\circ∘ 满足结合律。
  • 意义：满足结合律的运算，在连续运算时可以省略括号。

(3) 幂等律 (Idempotent Law)

  • 定义：如果对于 SSS 中任意元素 xxx，都有 x∘x=xx \circ x = xx∘x=x，则称 ∘\circ∘ 满足幂等律。

(4) 分配律 (Distributive Law)

  • 定义：涉及两个运算 ∘\circ∘ 和 ∗*∗。
    • 左分配律：x∗(y∘z)=(x∗y)∘(x∗z)x * (y \circ z) = (x * y) \circ (x * z)x∗(y∘z)=(x∗y)∘(x∗z)
    • 右分配律：(y∘z)∗x=(y∗x)∘(z∗x)(y \circ z) * x = (y * x) \circ (z * x)(y∘z)∗x=(y∗x)∘(z∗x)
  • 注意：如果左右分配律都成立，则称 ∗*∗ 对 ∘\circ∘ 是可分配的。（注意是谁对谁分配，通常是括号外的对括号内的分配）。

(5) 吸收律 (Absorption Law)

  • 定义：涉及两个运算 ∘\circ∘ 和 ∗*∗。如果对于任意 x,yx, yx,y 都有：

    x∗(x∘y)=xx * (x \circ y) = x x∗(x∘y)=x

    x∘(x∗y)=xx \circ (x * y) = x x∘(x∗y)=x

    则称这两个运算满足吸收律。

3. 特殊元素（单位元、零元、逆元）

设 ∘\circ∘ 是集合 SSS 上的二元运算。

(1) 单位元 / 幺元 (Identity Element)

  • 左单位元：存在 el∈Se_l \in Sel​∈S，使得 ∀x∈S\forall x \in S∀x∈S，都有 el∘x=xe_l \circ x = xel​∘x=x。
  • 右单位元：存在 er∈Se_r \in Ser​∈S，使得 ∀x∈S\forall x \in S∀x∈S，都有 x∘er=xx \circ e_r = xx∘er​=x。
  • 单位元：若一个元素 eee 既是左单位元又是右单位元，则称 eee 为单位元。
  • 定理：如果单位元存在，则它一定是唯一的。

(2) 零元 (Zero Element)

  • 左零元：存在 θl∈S\theta_l \in Sθl​∈S，使得 ∀x∈S\forall x \in S∀x∈S，都有 θl∘x=θl\theta_l \circ x = \theta_lθl​∘x=θl​。
  • 右零元：存在 θr∈S\theta_r \in Sθr​∈S，使得 ∀x∈S\forall x \in S∀x∈S，都有 x∘θr=θrx \circ \theta_r = \theta_rx∘θr​=θr​。
  • 零元：若一个元素 θ\thetaθ 既是左零元又是右零元，则称 θ\thetaθ 为零元。
  • 定理：如果零元存在，则它一定是唯一的。

(3) 逆元 (Inverse Element)

  • 前提：运算必须有单位元 eee。
  • 左逆元：对于 x∈Sx \in Sx∈S，若存在 yly_lyl​，使得 yl∘x=ey_l \circ x = eyl​∘x=e，则称 yly_lyl​ 是 xxx 的左逆元。
  • 右逆元：对于 x∈Sx \in Sx∈S，若存在 yry_ryr​，使得 x∘yr=ex \circ y_r = ex∘yr​=e，则称 yry_ryr​ 是 xxx 的右逆元。
  • 逆元：若 yyy 既是 xxx 的左逆元又是右逆元，则称 yyy 是 xxx 的逆元，记作 x−1x^{-1}x−1（乘法表示）或 −x-x−x（加法表示）。
  • 定理：如果运算满足结合律，且元素 xxx 有逆元，则其逆元是唯一的。
  • 逆元的运算：若满足结合律，(x∘y)−1=y−1∘x−1(x \circ y)^{-1} = y^{-1} \circ x^{-1}(x∘y)−1=y−1∘x−1。

(4) 消去律 (Cancellation Law)

  • 定义：
    • 左消去律：若 x∘y=x∘zx \circ y = x \circ zx∘y=x∘z 且 x≠θx \neq \thetax=θ（非零元），则 y=zy = zy=z。
    • 右消去律：若 y∘x=z∘xy \circ x = z \circ xy∘x=z∘x 且 x≠θx \neq \thetax=θ，则 y=zy = zy=z。
  • 定理：如果一个运算是可交换的，且每个非零元素都有逆元，则该运算满足消去律。

9.2 代数系统

1. 代数系统的定义

  • 定义：非空集合 SSS 和 SSS 上定义的若干个代数运算 f1,f2,…,fkf_1, f_2, \dots, f_kf1​,f2​,…,fk​ 组成的系统称为代数系统。
  • 记作：V=<S,f1,f2,…,fk>V = <S, f_1, f_2, \dots, f_k>V=<S,f1​,f2​,…,fk​>。
    • SSS 称为载体。
    • fif_ifi​ 称为运算。
  • 代数常数：有时为了强调某些特定元素（如单位元、零元）的重要性，也会把它们列在系统中。

2. 同类型的代数系统

  • 定义：如果两个代数系统 V1V_1V1​ 和 V2V_2V2​ 满足以下条件，则称它们是同类型的：
    1. 运算的个数相同。
    2. 对应运算的元数相同（例如都是二元运算，或都是一元运算）。
    • 注意：同类型不代表性质相同，只代表结构形式相似。

3. 子代数 (Subalgebra)

  • 定义：设 V=<S,f1,…,fk>V = <S, f_1, \dots, f_k>V=<S,f1​,…,fk​> 是代数系统， BBB 是 SSS 的非空子集。如果 BBB 对 VVV 中的所有运算都是封闭的（即运算结果仍在 BBB 中），则称 <B,f1,…,fk><B, f_1, \dots, f_k><B,f1​,…,fk​> 是 VVV 的子代数。
  • 平凡子代数：
    • VVV 本身（最大子代数）。
    • 由代数常数（如单位元）构成的最小子代数。
  • 真子代数：除了 VVV 本身以外的子代数。
  • 生成子代数：由集合 SSS 的子集生成的最小子代数。

4. 积代数 (Product Algebra)

  • 定义：设有两个同类型的代数系统 V1=<A,∘>V_1 = <A, \circ>V1​=<A,∘> 和 V2=<B,∗>V_2 = <B, *>V2​=<B,∗>。可以在集合 A×BA \times BA×B 上定义一个新的运算 ⋅\cdot⋅，使得：

    <a1,b1>⋅<a2,b2>=<a1∘a2,b1∗b2><a_1, b_1> \cdot <a_2, b_2> = <a_1 \circ a_2, b_1 * b_2> <a1​,b1​>⋅<a2​,b2​>=<a1​∘a2​,b1​∗b2​>

    称 V=<A×B,⋅>V = <A \times B, \cdot>V=<A×B,⋅> 为 V1V_1V1​ 和 V2V_2V2​ 的积代数。
  • 性质：积代数通常能继承原子代数的性质（如结合律、交换律、单位元、零元、逆元等）。
    • 积代数的单位元是 <e1,e2><e_1, e_2><e1​,e2​>。
    • 积代数的零元是 <θ1,θ2><\theta_1, \theta_2><θ1​,θ2​>。

9.3 代数系统的同态与同构

1. 同态与同构的定义

设 V1=<A,∘>V_1 = <A, \circ>V1​=<A,∘> 和 V2=<B,∗>V_2 = <B, *>V2​=<B,∗> 是同类型的代数系统，f:A→Bf: A \to Bf:A→B 是一个函数。

  • 同态映射 (Homomorphism)：
    如果对于 AAA 中任意元素 x,yx, yx,y，都有：

    f(x∘y)=f(x)∗f(y)f(x \circ y) = f(x) * f(y) f(x∘y)=f(x)∗f(y)

    则称 fff 是从 V1V_1V1​ 到 V2V_2V2​ 的同态映射。

    • 解释：先在 AAA 里运算再映射，等于先映射到 BBB 里再运算。运算结构被保持了。
    • 记作 V1∼V2V_1 \sim V_2V1​∼V2​。

  • 同态的分类：

    1. 单同态：fff 是单射。
    2. 满同态：fff 是满射（此时 V2V_2V2​ 称为 V1V_1V1​ 的同态像）。
    3. 同构 (Isomorphism)：fff 是双射（既单又满）。
      • 记作 V1≅V2V_1 \cong V_2V1​≅V2​。
      • 意义：两个同构的代数系统在抽象结构上是完全一样的，只是元素的名字不同。

  • 自同态与自同构：

    • 自同态：从 VVV 到 VVV 自身的同态。
    • 自同构：从 VVV 到 VVV 自身的同构。

2. 同态的性质

设 fff 是从 V1=<A,∘>V_1 = <A, \circ>V1​=<A,∘> 到 V2=<B,∗>V_2 = <B, *>V2​=<B,∗> 的满同态。

  1. 运算律的保持：
    • 若 ∘\circ∘ 满足交换律，则 ∗*∗ 也满足交换律。
    • 若 ∘\circ∘ 满足结合律，则 ∗*∗ 也满足结合律。
    • 若 ∘\circ∘ 满足幂等律，则 ∗*∗ 也满足幂等律。
  2. 特殊元素的保持：
    • 单位元：f(eA)=eBf(e_A) = e_Bf(eA​)=eB​ （AAA 的单位元映射过去是 BBB 的单位元）。
    • 零元：f(θA)=θBf(\theta_A) = \theta_Bf(θA​)=θB​。
    • 逆元：f(x−1)=(f(x))−1f(x^{-1}) = (f(x))^{-1}f(x−1)=(f(x))−1 （xxx 的逆元的像，就是 xxx 的像的逆元）。

第10章 群与环

10.1 群的定义及性质

1. 基本定义

  • 半群 (Semigroup)：
    • 设 V=<S,∘>V = <S, \circ>V=<S,∘> 是代数系统，如果运算 ∘\circ∘ 是可结合的（满足结合律），则称 VVV 为半群。
  • 独异点 (Monoid) / 幺半群：
    • 设 V=<S,∘>V = <S, \circ>V=<S,∘> 是半群，若 SSS 中存在关于 ∘\circ∘ 运算的单位元 eee，则称 VVV 是独异点。
    • 记作 V=<S,∘,e>V = <S, \circ, e>V=<S,∘,e>。
  • 群 (Group)：
    • 设 V=<S,∘>V = <S, \circ>V=<S,∘> 是独异点（即满足结合律且有单位元 eee），若 SSS 中的每个元素 aaa 都有逆元 a−1a^{-1}a−1（即 a∘a−1=a−1∘a=ea \circ a^{-1} = a^{-1} \circ a = ea∘a−1=a−1∘a=e），则称 VVV 是群。
    • 通常将群记作 GGG。

总结：群必须满足的4个条件

  1. 封闭性（代数系统隐含条件）。
  2. 结合律：(a∘b)∘c=a∘(b∘c)(a \circ b) \circ c = a \circ (b \circ c)(a∘b)∘c=a∘(b∘c)。
  3. 单位元：存在 eee，使得 a∘e=e∘a=aa \circ e = e \circ a = aa∘e=e∘a=a。
  4. 逆元：对任意 aaa，存在 a−1a^{-1}a−1，使得 a∘a−1=a−1∘a=ea \circ a^{-1} = a^{-1} \circ a = ea∘a−1=a−1∘a=e。

2. 特殊的群

  • 有限群与无限群：若群 GGG 的基数（元素个数）有限，称为有限群；否则称为无限群。
    • 群的阶：群 GGG 的基数 ∣G∣|G|∣G∣ 称为群的阶。
  • 交换群 / 阿贝尔群 (Abelian Group)：
    • 如果群 GGG 中的运算 ∘\circ∘ 满足交换律（即 a∘b=b∘aa \circ b = b \circ aa∘b=b∘a），则称 GGG 为交换群。
  • 平凡群：只含单位元 {e}\{e\}{e} 的群。
  • Klein 四元群：一个特殊的4阶交换群，运算特点是每个元素的逆元是它自己。

3. 元素的阶 (Order of an Element)

  • 定义：设 GGG 是群， a∈Ga \in Ga∈G。使得 ak=ea^k = eak=e 成立的最小正整数 kkk，称为元素 aaa 的阶（或周期）。
    • 记作 ∣a∣=k|a| = k∣a∣=k。
  • 无限阶：若不存在这样的正整数 kkk，则称 aaa 为无限阶元。
  • 性质：
    • 单位元 eee 的阶是 1。
    • ∣a∣=∣a−1∣|a| = |a^{-1}|∣a∣=∣a−1∣。
    • ak=e  ⟺  ∣a∣a^k = e \iff |a|ak=e⟺∣a∣ 整除 kkk。

4. 群的性质

设 GGG 为群：

  1. 逆元唯一性：每个元素的逆元是唯一的。
  2. 消去律成立：ab=ac⇒b=cab = ac \Rightarrow b = cab=ac⇒b=c；ba=ca⇒b=cba = ca \Rightarrow b = cba=ca⇒b=c。
    • 注意：一般代数系统不一定有消去律，但群一定有。
  3. 方程有唯一解：方程 ax=bax=bax=b 和 ya=bya=bya=b 在群中必有唯一解（x=a−1b,y=ba−1x=a^{-1}b, y=ba^{-1}x=a−1b,y=ba−1）。
  4. 幂运算规则：
    • (a−1)−1=a(a^{-1})^{-1} = a(a−1)−1=a
    • (ab)−1=b−1a−1(ab)^{-1} = b^{-1}a^{-1}(ab)−1=b−1a−1 （穿脱鞋原理：顺序要颠倒）
    • 如果 GGG 是交换群，则 (ab)n=anbn(ab)^n = a^n b^n(ab)n=anbn；否则一般不成立。

10.2 子群与群的陪集分解

1. 子群 (Subgroup)

  • 定义：设 GGG 是群， HHH 是 GGG 的非空子集。如果 HHH 关于 GGG 中的运算也构成群，则称 HHH 是 GGG 的子群。
    • 记作 H≤GH \le GH≤G。若 H≠GH \neq GH=G，则称 HHH 是真子群 (H<GH < GH<G)。
  • 平凡子群：{e}\{e\}{e} 和 GGG 本身。
  • 判定定理：
    1. 判定定理一：H≤G  ⟺  ∀a,b∈H,ab∈HH \le G \iff \forall a, b \in H, a b \in HH≤G⟺∀a,b∈H,ab∈H 且 a−1∈Ha^{-1} \in Ha−1∈H。（封闭且有逆元）
    2. 判定定理二：H≤G  ⟺  ∀a,b∈H,ab−1∈HH \le G \iff \forall a, b \in H, a b^{-1} \in HH≤G⟺∀a,b∈H,ab−1∈H。（一步判定法）
    3. 有穷子群判定：若 HHH 是群 GGG 的有穷非空子集，只要 HHH 对运算封闭，则 HHH 是子群。（有穷集蕴含逆元存在）

2. 生成子群与循环群

  • 生成子群：设 a∈Ga \in Ga∈G，由 aaa 的所有幂 ak(k∈Z)a^k (k \in \mathbb{Z})ak(k∈Z) 构成的集合，是 GGG 的一个子群，称为由 aaa 生成的子群，记作 <a><a\hspace{-2pt}><a>。
    • <a>={ak∣k∈Z}<a\hspace{-2pt}> = \{ a^k \mid k \in \mathbb{Z} \}<a>={ak∣k∈Z}。
  • 循环群 (Cyclic Group)：
    • 如果群 GGG 中存在一个元素 aaa，使得 G=<a>G = <a\hspace{-2pt}>G=<a>，则称 GGG 为循环群。
    • aaa 称为 GGG 的生成元。
    • 例子：
      • 整数加法群 <Z,+><\mathbb{Z}, +><Z,+> 是循环群，生成元是 111 和 −1-1−1。
      • 模 nnn 加法群 <Zn,⊕><\mathbb{Z}_n, \oplus><Zn​,⊕> 是循环群，生成元是 111 等。
    • 性质：循环群一定是交换群。

3. 陪集 (Coset)

  • 定义：设 HHH 是 GGG 的子群， a∈Ga \in Ga∈G。
    • 左陪集：aH={ah∣h∈H}aH = \{ ah \mid h \in H \}aH={ah∣h∈H}。
    • 右陪集：Ha={ha∣h∈H}Ha = \{ ha \mid h \in H \}Ha={ha∣h∈H}。
    • aaa 称为该陪集的代表元素。
  • 性质：
    • 陪集之间要么完全相等，要么互不相交（构成了 GGG 的一个划分）。
    • ∣aH∣=∣Ha∣=∣H∣|aH| = |Ha| = |H|∣aH∣=∣Ha∣=∣H∣ （陪集的大小等于子群的大小）。
    • a∈H  ⟺  aH=Ha \in H \iff aH = Ha∈H⟺aH=H。

4. 拉格朗日定理 (Lagrange’s Theorem)

  • 内容：设 GGG 是有限群， HHH 是 GGG 的子群。则 HHH 的阶 ∣H∣|H|∣H∣ 整除 GGG 的阶 ∣G∣|G|∣G∣。

    ∣G∣=∣H∣⋅[G:H]|G| = |H| \cdot [G:H] ∣G∣=∣H∣⋅[G:H]

    • [G:H][G:H][G:H] 称为 HHH 在 GGG 中的指数（即不同的左陪集个数）。
  • 推论：
    1. 有限群中，任意元素的阶 ∣a∣|a|∣a∣ 整除群的阶 ∣G∣|G|∣G∣。
    2. 素数阶的群一定是循环群。
    3. 若 ∣G∣=n|G|=n∣G∣=n，则 ∀a∈G,an=e\forall a \in G, a^n = e∀a∈G,an=e。

5. 正规子群 (Normal Subgroup)

  • 定义：设 HHH 是 GGG 的子群。如果对于任意 a∈Ga \in Ga∈G，都有 aH=HaaH = HaaH=Ha（左陪集等于右陪集），则称 HHH 为 GGG 的正规子群（或不变子群）。
    • 记作 H⊴GH \trianglelefteq GH⊴G。
  • 判定：
    • H⊴G  ⟺  ∀g∈G,∀h∈H,ghg−1∈HH \trianglelefteq G \iff \forall g \in G, \forall h \in H, g h g^{-1} \in HH⊴G⟺∀g∈G,∀h∈H,ghg−1∈H。
    • 交换群的所有子群都是正规子群。

6. 商群 (Quotient Group)

  • 定义：设 HHH 是 GGG 的正规子群。由 HHH 的所有陪集构成的集合 G/H={aH∣a∈G}G/H = \{ aH \mid a \in G \}G/H={aH∣a∈G}，在定义运算 (aH)(bH)=(ab)H(aH)(bH) = (ab)H(aH)(bH)=(ab)H 下构成一个群，称为商群。
  • 阶：∣G/H∣=∣G∣/∣H∣|G/H| = |G| / |H|∣G/H∣=∣G∣/∣H∣。

10.3 循环群与置换群

1. 循环群的深入性质

  • 分类：
    • 无限循环群：同构于整数加法群 <Z,+><\mathbb{Z}, +><Z,+>。
    • nnn 阶有限循环群：同构于模 nnn 加法群 <Zn,⊕><\mathbb{Z}_n, \oplus><Zn​,⊕>。
  • 子群结构：
    • 循环群的子群都是循环群。
    • 对于有限循环群 GGG（阶为 nnn），对于 nnn 的每个正因子 ddd，恰好存在一个 ddd 阶子群。

2. 置换群 (Permutation Group)

  • 置换：集合 S={1,2,…,n}S = \{1, 2, \dots, n\}S={1,2,…,n} 上的双射函数 σ:S→S\sigma: S \to Sσ:S→S 称为 SSS 上的 nnn 元置换。
  • 对称群 SnS_nSn​：SSS 上所有 n!n!n! 个置换构成的群（运算是函数的复合），称为 nnn 次对称群。
  • 置换的表示：
    • 轮换 (Cycle)：(i1i2…ik)(i_1 i_2 \dots i_k)(i1​i2​…ik​) 表示 i1→i2→⋯→ik→i1i_1 \to i_2 \to \dots \to i_k \to i_1i1​→i2​→⋯→ik​→i1​，其余元素不变。
    • 任何置换都可以分解为不相交轮换的乘积，且分解形式唯一。
  • 对换 (Transposition)：长度为 2 的轮换 (ij)(i j)(ij)。
    • 任何置换都可以分解为对换的乘积。
    • 奇偶性：分解成奇数个对换称为奇置换，偶数个称为偶置换。
  • 交错群 AnA_nAn​：SnS_nSn​ 中所有偶置换构成的子群，阶为 n!/2n!/2n!/2。
  • 凯莱定理 (Cayley’s Theorem)：任何群都同构于一个置换群（通常是它自身的左平移置换群）。

10.4 环与域

本节将代数系统从“一个运算”扩展到“两个运算”（通常称为加法 +++ 和乘法 ⋅\cdot⋅）。

1. 环 (Ring)

  • 定义：代数系统 <R,+,⋅><R, +, \cdot><R,+,⋅> 称为环，如果满足：
    1. <R,+><R, +><R,+> 是交换群（阿贝尔群）。
    2. <R,⋅><R, \cdot><R,⋅> 是半群（封闭、结合律）。
    3. 乘法对加法满足分配律：a(b+c)=ab+aca(b+c) = ab + aca(b+c)=ab+ac。
  • 基本性质：
    • 零元 000（加法单位元）是乘法零元：a⋅0=0⋅a=0a \cdot 0 = 0 \cdot a = 0a⋅0=0⋅a=0。
    • 符号规则：(−a)b=a(−b)=−(ab)(-a)b = a(-b) = -(ab)(−a)b=a(−b)=−(ab)。

2. 特殊的环

  • 交换环：乘法满足交换律 (ab=baab = baab=ba)。
  • 含幺环：乘法有单位元 111。
  • 无零因子环：若 ab=0⇒a=0∨b=0ab=0 \Rightarrow a=0 \lor b=0ab=0⇒a=0∨b=0。
    • 零因子：非零元素 a,ba, ba,b 若满足 ab=0ab=0ab=0，则称它们为零因子。
  • 整环 (Integral Domain)：
    • 定义：交换、含幺、无零因子的环。
    • 例子：整数环 <Z,+,⋅><\mathbb{Z}, +, \cdot><Z,+,⋅> 是最典型的整环。
    • 性质：整环中满足乘法消去律（若 ab=acab=acab=ac 且 a≠0a \neq 0a=0，则 b=cb=cb=c）。

3. 域 (Field)

  • 定义：设 <F,+,⋅><F, +, \cdot><F,+,⋅> 是一个交换环（有单位元 1≠01 \neq 01=0）。如果 FFF 中每个非零元素都有乘法逆元，则称 FFF 为域。
    • 简单说：域是 <F,+><F, +><F,+> 为交换群，且 <F−{0},⋅><F-\{0\}, \cdot><F−{0},⋅> 也为交换群的系统。
  • 例子：
    • 有理数域 Q\mathbb{Q}Q、实数域 R\mathbb{R}R、复数域 C\mathbb{C}C。
    • 有限域 (Galois Field)：模素数 ppp 的整数环 <Zp,+,⋅><\mathbb{Z}_p, +, \cdot><Zp​,+,⋅> 是一个域。
  • 关系：
    • 域 ⇒\Rightarrow⇒ 整环 ⇒\Rightarrow⇒ 环。
    • 有限整环一定是域。

10.5 典型应用示例（简述）

  • RSA公钥密码：基于大数分解的困难性。利用了欧拉定理（群论性质）mϕ(n)≡1(modn)m^{\phi(n)} \equiv 1 \pmod nmϕ(n)≡1(modn)。
  • 纠错码：利用有限域上的线性代数结构进行编码和解码。

第11章 格与布尔代数

11.1 格的定义与性质

格（Lattice）既可以看作是一种特殊的偏序集，也可以看作是一种具有两个二元运算的代数系统。

1. 格作为偏序集的定义

  • 定义：设 <S,≤><S, \le><S,≤> 是偏序集。如果对于 SSS 中的任意两个元素 xxx 和 yyy，它们的集合 {x,y}\{x, y\}{x,y} 都存在最小上界和最大下界，则称 SSS 关于偏序 ≤\le≤ 构成一个格。
  • 记号：
    • 析取 (∨\lor∨) / 上确界：xxx 与 yyy 的最小上界记作 x∨yx \lor yx∨y （Join）。
    • 合取 (∧\land∧) / 下确界：xxx 与 yyy 的最大下界记作 x∧yx \land yx∧y （Meet）。
  • 注意：这里的 ∨\lor∨ 和 ∧\land∧ 是格中的运算符号，不要单纯理解为逻辑或/与，也不要理解为集合的并/交（除非是在幂集格中）。

2. 格作为代数系统的定义

  • 定义：设 <S,∗,∘><S, *, \circ><S,∗,∘> 是具有两个二元运算的代数系统。如果 ∗*∗ 和 ∘\circ∘ 满足以下四组定律，则称 SSS 为格。通常将 ∗*∗ 记为 ∧\land∧，∘\circ∘ 记为 ∨\lor∨。
    1. 交换律：

      a∧b=b∧a,a∨b=b∨aa \land b = b \land a, \quad a \lor b = b \lor a a∧b=b∧a,a∨b=b∨a

    2. 结合律：

      (a∧b)∧c=a∧(b∧c),(a∨b)∨c=a∨(b∨c)(a \land b) \land c = a \land (b \land c), \quad (a \lor b) \lor c = a \lor (b \lor c) (a∧b)∧c=a∧(b∧c),(a∨b)∨c=a∨(b∨c)

    3. 幂等律：

      a∧a=a,a∨a=aa \land a = a, \quad a \lor a = a a∧a=a,a∨a=a

    4. 吸收律 (最显著的特征)：

      a∧(a∨b)=a,a∨(a∧b)=aa \land (a \lor b) = a, \quad a \lor (a \land b) = a a∧(a∨b)=a,a∨(a∧b)=a

3. 两种定义的等价性

  • 由代数系统定义的格，可以诱导出一个偏序关系：

    a≤b⇔a∧b=a⇔a∨b=ba \le b \Leftrightarrow a \land b = a \Leftrightarrow a \lor b = b a≤b⇔a∧b=a⇔a∨b=b

    在该偏序下，原来的 ∧\land∧ 和 ∨\lor∨ 恰好就是最大下界和最小上界运算。
  • 结论：偏序集定义的格与代数系统定义的格是完全等价的。

4. 格的对偶原理

  • 定义：设 fff 是含有格中元素以及符号 =,≤,≥,∨,∧=, \le, \ge, \lor, \land=,≤,≥,∨,∧ 的命题。令 f∗f^*f∗ 是将 fff 中的 ≤\le≤ 替换成 ≥\ge≥，≥\ge≥ 替换成 ≤\le≤，∨\lor∨ 替换成 ∧\land∧，∧\land∧ 替换成 ∨\lor∨ 所得到的命题。称 f∗f^*f∗ 为 fff 的对偶命题。
  • 原理：若 fff 对一切格为真，则 f∗f^*f∗ 也对一切格为真。
    • 解释：格的定义在 ≤\le≤ 与 ≥\ge≥、∨\lor∨ 与 ∧\land∧ 之间是对称的。

5. 格的基本性质

设 LLL 是格，对于任意 a,b,c,d∈La, b, c, d \in La,b,c,d∈L：

  1. 不等式性质：

    a≤a∨b,b≤a∨ba \le a \lor b, \quad b \le a \lor b a≤a∨b,b≤a∨b

    a∧b≤a,a∧b≤ba \land b \le a, \quad a \land b \le b a∧b≤a,a∧b≤b

  2. 保序性 (Isotonicity)：
    • 若 b≤cb \le cb≤c，则 a∧b≤a∧ca \land b \le a \land ca∧b≤a∧c。
    • 若 b≤cb \le cb≤c，则 a∨b≤a∨ca \lor b \le a \lor ca∨b≤a∨c。
  3. 不等式传递：
    • 若 a≤ba \le ba≤b 且 c≤dc \le dc≤d，则 a∧c≤b∧da \land c \le b \land da∧c≤b∧d。
    • 若 a≤ba \le ba≤b 且 c≤dc \le dc≤d，则 a∨c≤b∨da \lor c \le b \lor da∨c≤b∨d。
  4. 分配不等式 (注意不是等式)：

    a∨(b∧c)≤(a∨b)∧(a∨c)a \lor (b \land c) \le (a \lor b) \land (a \lor c) a∨(b∧c)≤(a∨b)∧(a∨c)

    (a∧b)∨(a∧c)≤a∧(b∨c)(a \land b) \lor (a \land c) \le a \land (b \lor c) (a∧b)∨(a∧c)≤a∧(b∨c)

6. 子格

  • 定义：设 <L,∧,∨><L, \land, \lor><L,∧,∨> 是格， SSS 是 LLL 的非空子集。如果 SSS 关于 LLL 中的运算 ∧\land∧ 和 ∨\lor∨ 仍是封闭的，则称 SSS 为 LLL 的子格。
  • 注意：仅仅是 LLL 的子偏序集不一定是子格，必须保证子集中任意两元素的最小上界和最大下界还在子集中，且与原格中的结果一致。

11.2 分配格、有补格与布尔代数

1. 分配格 (Distributive Lattice)

  • 定义：如果格 LLL 中的运算 ∧\land∧ 和 ∨\lor∨ 满足分配律，则称 LLL 为分配格。

    a∧(b∨c)=(a∧b)∨(a∧c)a \land (b \lor c) = (a \land b) \lor (a \land c) a∧(b∨c)=(a∧b)∨(a∧c)

    a∨(b∧c)=(a∨b)∧(a∨c)a \lor (b \land c) = (a \lor b) \land (a \lor c) a∨(b∧c)=(a∨b)∧(a∨c)

    • 注：只要其中一个分配律成立，另一个必然成立。
  • 判定条件：
    格 LLL 是分配格 当且仅当 LLL 中不含有与 钻石格 (M5M_5M5​) 或 五角格 (N5N_5N5​) 同构的子格。
    • 钻石格 (M5M_5M5​)：5个元素，中间3个元素互不可比。
    • 五角格 (N5N_5N5​)：5个元素，形状像五边形（不规则）。

2. 有界格 (Bounded Lattice)

  • 全下界 (0)：若存在 0∈L0 \in L0∈L，使得 ∀x∈L,0≤x\forall x \in L, 0 \le x∀x∈L,0≤x。
  • 全上界 (1)：若存在 1∈L1 \in L1∈L，使得 ∀x∈L,x≤1\forall x \in L, x \le 1∀x∈L,x≤1。
  • 有界格：既有全下界 000 又有全上界 111 的格。
    • 记作 <L,∧,∨,0,1><L, \land, \lor, 0, 1><L,∧,∨,0,1>。
  • 性质：

    a∧0=0,a∨0=aa \land 0 = 0, \quad a \lor 0 = a a∧0=0,a∨0=a

    a∧1=a,a∨1=1a \land 1 = a, \quad a \lor 1 = 1 a∧1=a,a∨1=1

3. 有补格 (Complemented Lattice)

  • 补元：在有界格 LLL 中，对于 a∈La \in La∈L，若存在 b∈Lb \in Lb∈L 使得：

    a∧b=0且a∨b=1a \land b = 0 \quad \text{且} \quad a \lor b = 1 a∧b=0且a∨b=1

    则称 bbb 为 aaa 的补元（或补）。
  • 有补格：如果有界格 LLL 中的每一个元素都至少存在一个补元，则称 LLL 为有补格。
  • 唯一性定理：在分配格中，如果元素存在补元，则补元一定是唯一的。

4. 布尔代数 (Boolean Algebra)

  • 定义：一个格如果既是分配格，又是有补格，则称该格为布尔代数（或布尔格）。
  • 记作：<B,∧,∨,′,0,1><B, \land, \lor, ', 0, 1><B,∧,∨,′,0,1>。其中 ′'′ 表示求补运算。
  • 性质（除了满足格和分配格的所有性质外）：
    1. 互补律：a∧a′=0,a∨a′=1a \land a' = 0, \quad a \lor a' = 1a∧a′=0,a∨a′=1。
    2. 双重否定律：(a′)′=a(a')' = a(a′)′=a。
    3. 德·摩根律 (De Morgan’s Laws)：

      (a∧b)′=a′∨b′(a \land b)' = a' \lor b' (a∧b)′=a′∨b′

      (a∨b)′=a′∧b′(a \lor b)' = a' \land b' (a∨b)′=a′∧b′

5. 布尔代数的代数系统定义

布尔代数也可以直接定义为代数系统 <B,∗,∘,′,0,1><B, *, \circ, ', 0, 1><B,∗,∘,′,0,1>，满足：

  1. 交换律
  2. 分配律
  3. 同一律 (存在单位元 000 和 111)
  4. 补元律

6. 原子与有限布尔代数的结构

  • 原子 (Atom)：
    在布尔代数 BBB 中，元素 aaa 称为原子，如果 a≠0a \neq 0a=0，且不存在 b∈Bb \in Bb∈B 使得 0<b<a0 < b < a0<b<a。
    • 几何意义：哈斯图中紧挨着 000 上方的元素。
  • 表示定理：
    任何有限布尔代数 BBB 都同构于其所有原子构成的集合的幂集代数。
    • 推论1：有限布尔代数的基数（元素个数）一定是 222 的幂（2n2^n2n，其中 nnn 是原子个数）。
    • 推论2：任意两个基数相同的有限布尔代数都是同构的。
