第6章 集合代数

6.1 集合的基本概念

1. 集合与元素

  • 集合 (Set)：把一些事物汇集到一起组成的一个整体。集合本身是一个不能精确定义的基本概念。
  • 元素 (Element)：组成集合的这些事物。
  • 记号：
    • 集合通常用大写字母表示（如 A,B,CA, B, CA,B,C）。
    • 元素通常用小写字母表示（如 a,b,ca, b, ca,b,c）。
  • 隶属关系：
    • 属于：若 aaa 是集合 AAA 的元素，记作 a∈Aa \in Aa∈A。
    • 不属于：若 aaa 不是集合 AAA 的元素，记作 a∉Aa \notin Aa∈/A。

2. 集合的表示法

  1. 列元素法：将集合的所有元素列出来，用逗号分隔，花括号括起来。
    • 例如：A={a,b,c}A = \{a, b, c\}A={a,b,c}。
  2. 谓词表示法：用谓词来概括集合中元素的属性。
    • 格式：B={x∣P(x)}B = \{ x \mid P(x) \}B={x∣P(x)}。
    • 读作：BBB 是所有满足性质 P(x)P(x)P(x) 的 xxx 组成的集合。

3. 集合的特性

  • 互异性：集合中的元素是彼此不同的。如果列举时重复出现（如 {1,2,2}\{1, 2, 2\}{1,2,2}），应认为是一个元素（即 {1,2}\{1, 2\}{1,2}）。
  • 无序性：集合中的元素没有顺序（如 {1,2,3}={3,1,2}\{1, 2, 3\} = \{3, 1, 2\}{1,2,3}={3,1,2}）。
  • 元素可以是集合：一个集合的元素也可以是另一个集合。
    • 注意区分：a∈{a}a \in \{a\}a∈{a} 是成立的，但 a={a}a = \{a\}a={a} 通常不成立。

4. 集合间的关系

  • 子集 (Subset)：

    • 定义：如果集合 BBB 的每一个元素都是集合 AAA 的元素，则称 BBB 是 AAA 的子集。
    • 记作：B⊆AB \subseteq AB⊆A （读作 BBB 包含于 AAA，或 AAA 包含 BBB）。
    • 符号语言：B⊆A⇔∀x(x∈B→x∈A)B \subseteq A \Leftrightarrow \forall x (x \in B \rightarrow x \in A)B⊆A⇔∀x(x∈B→x∈A)。
    • 性质：任何集合都是其自身的子集（A⊆AA \subseteq AA⊆A）。

  • 相等 (Equality)：

    • 定义：如果 AAA 和 BBB 的元素完全相同。
    • 判定：AAA 是 BBB 的子集，且 BBB 也是 AAA 的子集。
    • 符号语言：A=B⇔(A⊆B∧B⊆A)A = B \Leftrightarrow (A \subseteq B \land B \subseteq A)A=B⇔(A⊆B∧B⊆A)。

  • 真子集 (Proper Subset)：

    • 定义：如果 BBB 是 AAA 的子集，并且 AAA 中至少有一个元素不属于 BBB（即 A≠BA \neq BA=B），则称 BBB 是 AAA 的真子集。
    • 记作：B⊂AB \subset AB⊂A。
    • 符号语言：B⊂A⇔(B⊆A∧B≠A)B \subset A \Leftrightarrow (B \subseteq A \land B \neq A)B⊂A⇔(B⊆A∧B=A)。

5. 特殊集合

  • 空集 (Empty Set)：

    • 定义：不含任何元素的集合。
    • 记作：∅\varnothing∅。
    • 性质：空集是一切集合的子集（∅⊆A\varnothing \subseteq A∅⊆A）。空集是唯一的。
    • 注意：{∅}\{\varnothing\}{∅} 不是空集，它是一个含有一个元素（该元素是空集）的集合。

  • 全集 (Universal Set)：

    • 定义：在特定研究范围内，包含所有讨论对象的集合。
    • 记作：EEE （有的教材记作 UUU）。
    • 性质：对于任何集合 AAA，都有 A⊆EA \subseteq EA⊆E。

  • 幂集 (Power Set)：

    • 定义：设 AAA 为集合，把 AAA 的全体子集构成的集合称为 AAA 的幂集。
    • 记作：P(A)P(A)P(A) 或 2A2^A2A。
    • 符号语言：P(A)={x∣x⊆A}P(A) = \{ x \mid x \subseteq A \}P(A)={x∣x⊆A}。
    • 元素个数：若集合 AAA 有 nnn 个元素（∣A∣=n|A|=n∣A∣=n），则 P(A)P(A)P(A) 有 2n2^n2n 个元素。
    • 例子：若 A={1,2}A=\{1, 2\}A={1,2}，则 P(A)={∅,{1},{2},{1,2}}P(A) = \{ \varnothing, \{1\}, \{2\}, \{1, 2\} \}P(A)={∅,{1},{2},{1,2}}。

6.2 集合的运算

1. 基本运算定义

设 A,BA, BA,B 为两个集合：

  1. 并集 (Union)

    • 定义：由属于 AAA 或者 属于 BBB 的所有元素组成的集合。
    • 符号：A∪BA \cup BA∪B
    • 公式：A∪B={x∣x∈A∨x∈B}A \cup B = \{ x \mid x \in A \lor x \in B \}A∪B={x∣x∈A∨x∈B}

  2. 交集 (Intersection)

    • 定义：由既属于 AAA 又 属于 BBB 的所有公共元素组成的集合。
    • 符号：A∩BA \cap BA∩B
    • 公式：A∩B={x∣x∈A∧x∈B}A \cap B = \{ x \mid x \in A \land x \in B \}A∩B={x∣x∈A∧x∈B}
    • 不交：若 A∩B=∅A \cap B = \varnothingA∩B=∅，则称 AAA 与 BBB 不交（或互不相容）。

  3. 相对补集 / 差集 (Difference)

    • 定义：由属于 AAA 但不属于 BBB 的元素组成的集合。
    • 符号：A−BA - BA−B （有时记作 A∖BA \setminus BA∖B）
    • 公式：A−B={x∣x∈A∧x∉B}A - B = \{ x \mid x \in A \land x \notin B \}A−B={x∣x∈A∧x∈/B}

  4. 绝对补集 (Complement)

    • 定义：相对于全集 EEE，所有不属于 AAA 的元素组成的集合。
    • 符号：∼A\sim A∼A （或 AcA^cAc, A‾\overline{A}A）
    • 公式：∼A=E−A={x∣x∈E∧x∉A}\sim A = E - A = \{ x \mid x \in E \land x \notin A \}∼A=E−A={x∣x∈E∧x∈/A}

  5. 对称差集 (Symmetric Difference)

    • 定义：属于 AAA 或属于 BBB，但不一样时属于两者的元素集合（即“非公有”元素）。
    • 符号：A⊕BA \oplus BA⊕B
    • 公式：
      • 定义1：A⊕B=(A−B)∪(B−A)A \oplus B = (A - B) \cup (B - A)A⊕B=(A−B)∪(B−A)
      • 定义2：A⊕B=(A∪B)−(A∩B)A \oplus B = (A \cup B) - (A \cap B)A⊕B=(A∪B)−(A∩B)

2. 广义并与广义交

当有多个集合（甚至无穷多个）进行运算时：

  • 广义并 (⋃\bigcup⋃)：只要元素属于至少一个集合，就属于广义并。
    • ⋃i=1nAi=A1∪A2∪⋯∪An={x∣∃i(1≤i≤n∧x∈Ai)}\bigcup_{i=1}^{n} A_i = A_1 \cup A_2 \cup \dots \cup A_n = \{ x \mid \exists i (1 \le i \le n \land x \in A_i) \}⋃i=1n​Ai​=A1​∪A2​∪⋯∪An​={x∣∃i(1≤i≤n∧x∈Ai​)}
  • 广义交 (⋂\bigcap⋂)：元素必须属于每一个集合，才属于广义交。
    • ⋂i=1nAi=A1∩A2∩⋯∩An={x∣∀i(1≤i≤n→x∈Ai)}\bigcap_{i=1}^{n} A_i = A_1 \cap A_2 \cap \dots \cap A_n = \{ x \mid \forall i (1 \le i \le n \rightarrow x \in A_i) \}⋂i=1n​Ai​=A1​∩A2​∩⋯∩An​={x∣∀i(1≤i≤n→x∈Ai​)}

6.3 有穷集的计数

1. 文氏图 (Venn Diagram)

  • 用平面上的封闭曲线（通常是圆）内部区域表示集合。
  • 用矩形表示全集 EEE。
  • 用于直观地表示集合间的关系和运算结果。

2. 包含排斥原理 (Inclusion-Exclusion Principle)

用于计算多个集合并集的元素个数（基数）。记 ∣A∣|A|∣A∣ 为集合 AAA 的元素个数。

  • 两个集合：

    ∣A∪B∣=∣A∣+∣B∣−∣A∩B∣|A \cup B| = |A| + |B| - |A \cap B| ∣A∪B∣=∣A∣+∣B∣−∣A∩B∣

    • 解释：把 AAA 加一遍，BBB 加一遍，中间重叠的部分（交集）被加了两次，所以要减去一次。

  • 三个集合：

    ∣A∪B∪C∣=∣A∣+∣B∣+∣C∣−(∣A∩B∣+∣B∩C∣+∣C∩A∣)+∣A∩B∩C∣|A \cup B \cup C| = |A| + |B| + |C| - (|A \cap B| + |B \cap C| + |C \cap A|) + |A \cap B \cap C| ∣A∪B∪C∣=∣A∣+∣B∣+∣C∣−(∣A∩B∣+∣B∩C∣+∣C∩A∣)+∣A∩B∩C∣

    • 解释：加单个，减两两交，加三三交。

  • n 个集合（通用公式）：

    ∣⋃i=1nAi∣=∑∣Ai∣−∑∣Ai∩Aj∣+∑∣Ai∩Aj∩Ak∣−⋯+(−1)n−1∣A1∩⋯∩An∣|\bigcup_{i=1}^n A_i| = \sum |A_i| - \sum |A_i \cap A_j| + \sum |A_i \cap A_j \cap A_k| - \dots + (-1)^{n-1} |A_1 \cap \dots \cap A_n| ∣i=1⋃n​Ai​∣=∑∣Ai​∣−∑∣Ai​∩Aj​∣+∑∣Ai​∩Aj​∩Ak​∣−⋯+(−1)n−1∣A1​∩⋯∩An​∣

    • 规律：奇数个集合的交集取“+”，偶数个集合的交集取“-”。

6.4 集合恒等式

集合运算满足一系列代数定律，这些定律与命题逻辑的等值式非常相似。
设 A,B,CA, B, CA,B,C 为任意集合，EEE 为全集，∅\varnothing∅ 为空集。

1. 基本定律（必须熟记）

  1. 幂等律：

    A∪A=AA \cup A = A A∪A=A

    A∩A=AA \cap A = A A∩A=A

  2. 结合律：

    (A∪B)∪C=A∪(B∪C)(A \cup B) \cup C = A \cup (B \cup C) (A∪B)∪C=A∪(B∪C)

    (A∩B)∩C=A∩(B∩C)(A \cap B) \cap C = A \cap (B \cap C) (A∩B)∩C=A∩(B∩C)

  3. 交换律：

    A∪B=B∪AA \cup B = B \cup A A∪B=B∪A

    A∩B=B∩AA \cap B = B \cap A A∩B=B∩A

  4. 分配律 (极重要)：

    A∪(B∩C)=(A∪B)∩(A∪C)A \cup (B \cap C) = (A \cup B) \cap (A \cup C) A∪(B∩C)=(A∪B)∩(A∪C)

    A∩(B∪C)=(A∩B)∪(A∩C)A \cap (B \cup C) = (A \cap B) \cup (A \cap C) A∩(B∪C)=(A∩B)∪(A∩C)

  5. 同一律：

    A∪∅=AA \cup \varnothing = A A∪∅=A

    A∩E=AA \cap E = A A∩E=A

  6. 零律 (支配律)：

    A∪E=EA \cup E = E A∪E=E

    A∩∅=∅A \cap \varnothing = \varnothing A∩∅=∅

  7. 排中律 (互补律的一部分)：

    A∪∼A=EA \cup \sim A = E A∪∼A=E

  8. 矛盾律 (互补律的一部分)：

    A∩∼A=∅A \cap \sim A = \varnothing A∩∼A=∅

  9. 吸收律：

    A∪(A∩B)=AA \cup (A \cap B) = A A∪(A∩B)=A

    A∩(A∪B)=AA \cap (A \cup B) = A A∩(A∪B)=A

  10. 德·摩根律 (De Morgan’s Laws)：

    ∼(A∪B)=∼A∩∼B\sim (A \cup B) = \sim A \cap \sim B ∼(A∪B)=∼A∩∼B

    ∼(A∩B)=∼A∪∼B\sim (A \cap B) = \sim A \cup \sim B ∼(A∩B)=∼A∪∼B

    • 口诀：补集进括号，符号并变交，交变并。
  11. 双重否定律：

    ∼(∼A)=A\sim (\sim A) = A ∼(∼A)=A

2. 其他重要等式

  • 差集与补集的关系 (证明题常用)：

    A−B=A∩∼BA - B = A \cap \sim B A−B=A∩∼B

  • 包含关系的等价形式：
    以下几个式子是等价的：
    1. A⊆BA \subseteq BA⊆B
    2. A∩B=AA \cap B = AA∩B=A
    3. A∪B=BA \cup B = BA∪B=B
    4. A−B=∅A - B = \varnothingA−B=∅
    5. ∼B⊆∼A\sim B \subseteq \sim A∼B⊆∼A

3. 证明集合相等的方法

要证明 A=BA = BA=B，通常有以下几种方法：

  1. 双向包含法：分别证明 A⊆BA \subseteq BA⊆B 且 B⊆AB \subseteq AB⊆A。
  2. 命题演算法：利用 x∈A⇔⋯⇔x∈Bx \in A \Leftrightarrow \dots \Leftrightarrow x \in Bx∈A⇔⋯⇔x∈B，将集合运算转化为逻辑运算，利用逻辑等值式进行推导。
  3. 集合演算法：直接利用上述的“集合恒等式”进行代数变形。
  4. 隶属表法：类似于真值表，列出元素属于各个集合的所有可能情况（0表示不属于，1表示属于），验证两边结果是否一致。

第7章 二元关系

7.1 有序对与笛卡儿积

1. 有序对 (Ordered Pair)

  • 定义：由两个元素 xxx 和 yyy 按照一定顺序排列成的二元组。
  • 记作：⟨x,y⟩\langle x, y \rangle⟨x,y⟩。
    • xxx 是第一元素，yyy 是第二元素。
  • 性质：
    1. 有序性：如果 x≠yx \neq yx=y，那么 ⟨x,y⟩≠⟨y,x⟩\langle x, y \rangle \neq \langle y, x \rangle⟨x,y⟩=⟨y,x⟩。
    2. 相等条件：两个有序对相等，当且仅当对应的元素都相等。

      ⟨x,y⟩=⟨u,v⟩⇔(x=u∧y=v)\langle x, y \rangle = \langle u, v \rangle \Leftrightarrow (x = u \land y = v) ⟨x,y⟩=⟨u,v⟩⇔(x=u∧y=v)

2. 笛卡儿积 (Cartesian Product)

  • 定义：设 A,BA, BA,B 为集合，用 AAA 中元素为第一元素，BBB 中元素为第二元素，构成的所有可能的有序对的集合。
  • 记作：A×BA \times BA×B。
  • 公式：

    A×B={⟨x,y⟩∣x∈A∧y∈B}A \times B = \{ \langle x, y \rangle \mid x \in A \land y \in B \} A×B={⟨x,y⟩∣x∈A∧y∈B}

  • 基数（元素个数）：如果 ∣A∣=m,∣B∣=n|A| = m, |B| = n∣A∣=m,∣B∣=n，则 ∣A×B∣=m⋅n|A \times B| = m \cdot n∣A×B∣=m⋅n。
  • 性质：
    • 不满足交换律：A×B≠B×AA \times B \neq B \times AA×B=B×A （除非 A=BA=BA=B 或其中之一为空集）。
    • 不满足结合律：(A×B)×C≠A×(B×C)(A \times B) \times C \neq A \times (B \times C)(A×B)×C=A×(B×C)。
    • 对并/交运算满足分配律：

      A×(B∪C)=(A×B)∪(A×C)A \times (B \cup C) = (A \times B) \cup (A \times C) A×(B∪C)=(A×B)∪(A×C)

      A×(B∩C)=(A×B)∩(A×C)A \times (B \cap C) = (A \times B) \cap (A \times C) A×(B∩C)=(A×B)∩(A×C)

    • 零律：A×∅=∅×A=∅A \times \varnothing = \varnothing \times A = \varnothingA×∅=∅×A=∅。

7.2 二元关系

1. 关系的定义

  • 定义：一个集合如果满足以下条件之一，就称为二元关系（简称关系）：
    1. 集合非空，且它的元素都是有序对。
    2. 集合是空集 (∅\varnothing∅)。
  • 本质：关系 RRR 是笛卡儿积 A×BA \times BA×B 的一个子集 (R⊆A×BR \subseteq A \times BR⊆A×B)。
  • 记法：
    • 如果 ⟨x,y⟩∈R\langle x, y \rangle \in R⟨x,y⟩∈R，记作 xRyx R yxRy （读作 xxx 与 yyy 有关系 RRR）。
    • 如果 ⟨x,y⟩∉R\langle x, y \rangle \notin R⟨x,y⟩∈/R，记作 x̸Ryx \not\kern-0.2em R yxRy。

2. 特殊的关系类型

  • AAA 到 BBB 的关系：R⊆A×BR \subseteq A \times BR⊆A×B。
  • AAA 上的关系：R⊆A×AR \subseteq A \times AR⊆A×A （即起点和终点都在同一个集合 AAA 中）。
  • 关系的个数：如果 ∣A∣=n|A|=n∣A∣=n，则 AAA 上不同的二元关系共有 2n22^{n^2}2n2 个。

3. 几个特殊的“A上的关系”

设 AAA 为任意集合：

  1. 空关系：∅\varnothing∅ （没有任何元素有关系）。
  2. 全域关系：EA=A×AE_A = A \times AEA​=A×A （所有元素之间都有关系）。
  3. 恒等关系：IA={⟨x,x⟩∣x∈A}I_A = \{ \langle x, x \rangle \mid x \in A \}IA​={⟨x,x⟩∣x∈A} （每个元素只和自己有关系）。
    • 注意：恒等关系在矩阵中对应单位矩阵。

4. 关系的表示方法

除了集合表达式，还可以用直观的方式表示有限集合上的关系：

  1. 关系矩阵 (Relation Matrix) MRM_RMR​：
    • 用 000 和 111 组成的矩阵表示。
    • 若 xiRxjx_i R x_jxi​Rxj​，则第 iii 行第 jjj 列为 111，否则为 000。
  2. 关系图 (Relation Graph) GRG_RGR​：
    • 用点表示集合中的元素。
    • 若 xRyx R yxRy，则画一条从 xxx 指向 yyy 的有向边。

7.3 关系的运算

1. 基本集合运算

由于关系本质上是集合（有序对的集合），所以集合的所有运算（并 ∪\cup∪、交 ∩\cap∩、差 −-−、补 ∼\sim∼）都适用于关系。

2. 专门的关系运算

设 RRR 是二元关系：

(1) 域 (Domain/Range/Field)

  • 定义域 (dom R)：所有“出发点”构成的集合（有序对的第一元素）。

    domR={x∣∃y(⟨x,y⟩∈R)}\text{dom} R = \{ x \mid \exists y (\langle x, y \rangle \in R) \} domR={x∣∃y(⟨x,y⟩∈R)}

  • 值域 (ran R)：所有“终点”构成的集合（有序对的第二元素）。

    ranR={y∣∃x(⟨x,y⟩∈R)}\text{ran} R = \{ y \mid \exists x (\langle x, y \rangle \in R) \} ranR={y∣∃x(⟨x,y⟩∈R)}

  • 域 (fld R)：定义域和值域的并集。

    fldR=domR∪ranR\text{fld} R = \text{dom} R \cup \text{ran} R fldR=domR∪ranR

(2) 逆关系 (Inverse Relation)

  • 定义：将 RRR 中所有有序对的方向颠倒。
  • 符号：R−1R^{-1}R−1
  • 公式：

    R−1={⟨y,x⟩∣⟨x,y⟩∈R}R^{-1} = \{ \langle y, x \rangle \mid \langle x, y \rangle \in R \} R−1={⟨y,x⟩∣⟨x,y⟩∈R}

  • 性质：(R−1)−1=R(R^{-1})^{-1} = R(R−1)−1=R。

(3) 复合/合成 (Composition)

  • 定义：设 F,GF, GF,G 为关系，FFF 与 GGG 的合成 记作 F∘GF \circ GF∘G。
  • 公式（注意顺序）：

    F∘G={⟨x,y⟩∣∃t(⟨x,t⟩∈F∧⟨t,y⟩∈G)}F \circ G = \{ \langle x, y \rangle \mid \exists t (\langle x, t \rangle \in F \land \langle t, y \rangle \in G) \} F∘G={⟨x,y⟩∣∃t(⟨x,t⟩∈F∧⟨t,y⟩∈G)}

  • 通俗理解：先经过 FFF，再经过 GGG。
    • 类似于“搭乘联程航班”：xxx 通过 FFF 飞到中转站 ttt，然后从 ttt 通过 GGG 飞到 yyy。最终结果是 ⟨x,y⟩\langle x, y \rangle⟨x,y⟩。
    • 特别注意：本书的记法 F∘GF \circ GF∘G 是“左复合”，即 FFF 在前 GGG 在后。这与函数复合 g∘fg \circ fg∘f（先 fff 后 ggg）的习惯可能相反，请务必留意。
  • 性质：
    • 结合律：(F∘G)∘H=F∘(G∘H)(F \circ G) \circ H = F \circ (G \circ H)(F∘G)∘H=F∘(G∘H)。
    • 逆运算：(F∘G)−1=G−1∘F−1(F \circ G)^{-1} = G^{-1} \circ F^{-1}(F∘G)−1=G−1∘F−1 （穿脱鞋原理：倒序）。

(4) 限制与像

  • 限制：R↾AR \upharpoonright AR↾A 表示只保留 RRR 中出发点在集合 AAA 里的那些关系。
  • 像：R[A]R[A]R[A] 表示集合 AAA 中的元素通过关系 RRR 能到达的所有终点的集合。

(5) 关系的幂 (Powers)

  • 定义：关系 RRR 自身的多次合成。
    • R0=IAR^0 = I_AR0=IA​ （恒等关系）
    • R1=RR^1 = RR1=R
    • R2=R∘RR^2 = R \circ RR2=R∘R
    • Rn+1=Rn∘RR^{n+1} = R^n \circ RRn+1=Rn∘R
  • 直观意义：RnR^nRn 中的 ⟨x,y⟩\langle x, y \rangle⟨x,y⟩ 表示从 xxx 到 yyy 恰好存在一条长度为 nnn 的路径。

7.4 关系的性质

设 RRR 是集合 AAA 上的关系，以下是 RRR 可能具有的5种重要性质：

1. 自反性 (Reflexive)

  • 定义：每个元素都和自己有关系。
  • 公式：∀x(x∈A→⟨x,x⟩∈R)\forall x (x \in A \rightarrow \langle x, x \rangle \in R)∀x(x∈A→⟨x,x⟩∈R)。
  • 特征：
    • 关系矩阵：主对角线全为 111。
    • 关系图：每个顶点都有自环。

2. 反自反性 (Irreflexive)

  • 定义：没有任何元素和自己有关系。
  • 公式：∀x(x∈A→⟨x,x⟩∉R)\forall x (x \in A \rightarrow \langle x, x \rangle \notin R)∀x(x∈A→⟨x,x⟩∈/R)。
  • 特征：
    • 关系矩阵：主对角线全为 000。
    • 关系图：每个顶点都没有自环。
  • 注意：如果不满足自反，不一定就是反自反（可能有的有环，有的没环）。

3. 对称性 (Symmetric)

  • 定义：如果有 x→yx \to yx→y，就一定有 y→xy \to xy→x（双向奔赴）。
  • 公式：∀x∀y(⟨x,y⟩∈R→⟨y,x⟩∈R)\forall x \forall y (\langle x, y \rangle \in R \rightarrow \langle y, x \rangle \in R)∀x∀y(⟨x,y⟩∈R→⟨y,x⟩∈R)。
  • 特征：
    • 关系矩阵：关于主对角线对称 (M=MTM = M^TM=MT)。
    • 关系图：两点之间如果有边，必须是双向边（或无边）。

4. 反对称性 (Antisymmetric)

  • 定义：如果 xxx 和 yyy 互有关系，那它们必须是同一个元素。换句话说，两个不同的元素之间，不能同时存在双向关系（要么没有，要么只有单向）。
  • 公式：∀x∀y((⟨x,y⟩∈R∧⟨y,x⟩∈R)→x=y)\forall x \forall y ((\langle x, y \rangle \in R \land \langle y, x \rangle \in R) \rightarrow x = y)∀x∀y((⟨x,y⟩∈R∧⟨y,x⟩∈R)→x=y)。
  • 特征：
    • 关系矩阵：对于 i≠ji \neq ji=j，如果 mij=1m_{ij}=1mij​=1，则必须 mji=0m_{ji}=0mji​=0。
    • 关系图：两个不同顶点之间最多只有一条单向边。

5. 传递性 (Transitive)

  • 定义：如果 x→yx \to yx→y 且 y→zy \to zy→z，那么 x→zx \to zx→z（朋友的朋友是朋友）。
  • 公式：∀x∀y∀z((⟨x,y⟩∈R∧⟨y,z⟩∈R)→⟨x,z⟩∈R)\forall x \forall y \forall z ((\langle x, y \rangle \in R \land \langle y, z \rangle \in R) \rightarrow \langle x, z \rangle \in R)∀x∀y∀z((⟨x,y⟩∈R∧⟨y,z⟩∈R)→⟨x,z⟩∈R)。
  • 特征：
    • 关系矩阵：M2M^2M2 中的非零位置，在 MMM 中也是非零。
    • 关系图：如果存在 xxx 到 zzz 的间接路径，则必须直接有一条边 x→zx \to zx→z。

7.5 关系的闭包 (Closure)

1. 闭包的概念

  • 定义：如果关系 RRR 不具有某种性质（如自反性），我们希望添加最少的有序对，使其具有该性质。这个包含了 RRR 且具有该性质的最小关系，称为 RRR 的闭包。

2. 三种常见闭包及其求法

设 RRR 是 AAA 上的关系：

(1) 自反闭包 r(R)r(R)r(R)

  • 含义：让关系变自反。
  • 求法：给所有节点加上自环。
  • 公式：r(R)=R∪IAr(R) = R \cup I_Ar(R)=R∪IA​ （IAI_AIA​ 是恒等关系）。

(2) 对称闭包 s(R)s(R)s(R)

  • 含义：让关系变对称。
  • 求法：把所有单向边补成双向边。
  • 公式：s(R)=R∪R−1s(R) = R \cup R^{-1}s(R)=R∪R−1。

(3) 传递闭包 t(R)t(R)t(R)

  • 含义：让关系变传递。
  • 求法：如果 xxx 能通过任何路径到达 yyy，就直接连一条边 x→yx \to yx→y。
  • 公式：

    t(R)=R∪R2∪R3∪…t(R) = R \cup R^2 \cup R^3 \cup \dots t(R)=R∪R2∪R3∪…

  • 定理：如果集合 AAA 的元素个数为 nnn，则 t(R)=⋃i=1nRit(R) = \bigcup_{i=1}^{n} R^it(R)=⋃i=1n​Ri。
  • 算法：通常使用 Warshall 算法 在计算机中高效计算。

7.6 等价关系与划分

1. 等价关系 (Equivalence Relation)

  • 定义：如果一个关系 RRR 同时满足以下三个性质：
    1. 自反性
    2. 对称性
    3. 传递性
      则称 RRR 为等价关系。
  • 记作：x∼yx \sim yx∼y （表示 xxx 和 yyy 等价）。
  • 例子：“等于”、“同余”、“同姓”、“住在同一个城市”。

2. 等价类 (Equivalence Class)

  • 定义：在等价关系 RRR 下，与元素 xxx 有关系的所有元素组成的集合，称为 xxx 的等价类。
  • 记作：[x]R[x]_R[x]R​ 或简写为 [x][x][x]。
  • 公式：[x]R={y∣y∈A∧xRy}[x]_R = \{ y \mid y \in A \land x R y \}[x]R​={y∣y∈A∧xRy}。
  • 性质：
    • [x][x][x] 非空（至少包含 xxx 自己）。
    • 如果 xRyx R yxRy，则 [x]=[y][x] = [y][x]=[y] （它们的圈子完全一样）。
    • 如果 xxx 和 yyy 没关系，则 [x]∩[y]=∅[x] \cap [y] = \varnothing[x]∩[y]=∅ （圈子完全不沾边）。

3. 商集 (Quotient Set)

  • 定义：集合 AAA 上所有不同的等价类组成的集合。
  • 记作：A/RA/RA/R。
  • 形式：A/R={[x]R∣x∈A}A/R = \{ [x]_R \mid x \in A \}A/R={[x]R​∣x∈A}。
  • 例子：整数集 Z\mathbb{Z}Z 模3的同余关系，商集为 {{0,3,6… },{1,4,7… },{2,5,8… }}\{ \{0,3,6\dots\}, \{1,4,7\dots\}, \{2,5,8\dots\} \}{{0,3,6…},{1,4,7…},{2,5,8…}}。

4. 划分 (Partition)

  • 定义：把集合 AAA 分割成若干个非空子集，使得：
    1. 每个子集非空。
    2. 子集之间互不相交。
    3. 所有子集的并集等于 AAA。
  • 定理（核心）：
    • 等价关系诱导划分：任何等价关系的商集 A/RA/RA/R 就是 AAA 的一个划分。
    • 划分诱导等价关系：给定 AAA 的一个划分，一定存在唯一的等价关系，使得其等价类刚好就是这些划分块。
    • 结论：等价关系与划分是一一对应的。

7.7 偏序关系 (Partial Ordering)

1. 偏序关系的定义

  • 定义：如果一个关系 RRR 同时满足以下三个性质：
    1. 自反性
    2. 反对称性（注意：这里与等价关系不同，是对称 vs 反对称）
    3. 传递性
      则称 RRR 为偏序关系。
  • 记作：≤\le≤ （注意：这只是个符号，不一定代表数字大小）。
  • 偏序集 (Poset)：集合 AAA 和定义在它上面的偏序关系 ≤\le≤ 一起构成的二元组，记作 ⟨A,≤⟩\langle A, \le \rangle⟨A,≤⟩。
  • 例子：实数上的“小于等于 ≤\le≤”、集合间的“包含 ⊆\subseteq⊆”、正整数上的“整除”。

2. 可比性

  • 可比：对于 x,yx, yx,y，如果 x≤yx \le yx≤y 或 y≤xy \le xy≤x 成立，称 xxx 和 yyy 是可比的。
  • 全序关系 (Total Order)：如果集合中任意两个元素都是可比的，则称该关系为全序关系（或线序）。

3. 哈斯图 (Hasse Diagram)

一种简化的关系图，专门用于表示偏序集。

  • 画法规则：
    1. 省去自环（因为自反性是默认的）。
    2. 省去传递边（如果 x→y,y→zx \to y, y \to zx→y,y→z，则不画 x→zx \to zx→z）。
    3. 方向隐含：把图画成“下小上大”的层级结构，省去箭头。如果 x≤yx \le yx≤y，则 xxx 画在 yyy 的下方（直接或间接相连）。
  • 覆盖：如果 x≤yx \le yx≤y 且中间没有其他元素（即不存在 zzz 使得 x≤z≤yx \le z \le yx≤z≤y），称 yyy 覆盖 xxx。哈斯图中只画覆盖边。

4. 偏序集中的特殊元素

设 ⟨A,≤⟩\langle A, \le \rangle⟨A,≤⟩ 为偏序集， BBB 是 AAA 的子集。

(1) 极值元素（局部概念）

  • 极大元 (Maximal)：在 BBB 中，没有比它更大的元素（它是某个分支的顶端）。
    • 注意：极大元可以有多个。
  • 极小元 (Minimal)：在 BBB 中，没有比它更小的元素（它是某个分支的底端）。

(2) 最值元素（全局概念）

  • 最大元 (Maximum)：在 BBB 中，存在一个元素 bbb，使得 BBB 中所有其他元素都比它小 (x≤bx \le bx≤b)。
    • 注意：最大元如果存在，一定是唯一的。通常很多偏序集没有最大元（因为有多个山头）。
  • 最小元 (Minimum)：在 BBB 中，存在一个元素 bbb，使得 BBB 中所有其他元素都比它大 (b≤xb \le xb≤x)。

(3) 界（针对子集 B）

  • 上界 (Upper Bound)：AAA 中的元素 aaa，如果对于 BBB 中所有元素 xxx，都有 x≤ax \le ax≤a。
    • 最小上界 / 上确界 (Supremum/LUB)：所有上界中最小的那个。
  • 下界 (Lower Bound)：AAA 中的元素 aaa，如果对于 BBB 中所有元素 xxx，都有 a≤xa \le xa≤x。
    • 最大下界 / 下确界 (Infimum/GLB)：所有下界中最大的那个。

第8章 函数

8.1 函数的定义与性质

1. 函数的基本定义

函数是一种特殊的二元关系。

  • 定义：设 FFF 为二元关系，如果对于定义域 domF\text{dom}FdomF 中的每一个元素 xxx，都存在唯一的一个 y∈ranFy \in \text{ran}Fy∈ranF 使得 xFyxFyxFy（即 ⟨x,y⟩∈F\langle x, y \rangle \in F⟨x,y⟩∈F）成立，则称 FFF 为函数（或映射）。
  • 记作：y=F(x)y = F(x)y=F(x)。
  • 术语：yyy 称为 xxx 在 FFF 下的像（或值），xxx 称为 yyy 在 FFF 下的原像。
  • 核心特征：
    1. 单值性：一个输入只能对应一个输出（不能脚踏两只船）。
    2. 全定义性：定义域内的每个元素都必须有输出。

2. 函数的相等

由于函数是集合（有序对的集合），函数的相等即集合的相等。

  • 条件：两个函数 FFF 和 GGG 相等 (F=GF=GF=G)，当且仅当：
    1. 它们的定义域相同 (domF=domG\text{dom}F = \text{dom}GdomF=domG)。
    2. 对于定义域中的每一个 xxx，对应的函数值相同 (F(x)=G(x)F(x) = G(x)F(x)=G(x))。

3. 从 AAA 到 BBB 的函数

  • 定义：设 A,BA, BA,B 为集合，如果 fff 是函数，且满足：
    1. domf=A\text{dom}f = Adomf=A （定义域是 AAA 的全部）。
    2. ranf⊆B\text{ran}f \subseteq Branf⊆B （值域包含于 BBB）。
      则称 fff 是从 AAA 到 BBB 的函数。
  • 记作：f:A→Bf: A \to Bf:A→B。
  • 术语：
    • AAA 称为定义域 (Domain)。
    • BBB 称为陪域 (Codomain)。
    • f(A)={f(x)∣x∈A}f(A) = \{ f(x) \mid x \in A \}f(A)={f(x)∣x∈A} 称为值域 (Range)，它是 BBB 的子集。

4. 函数集合 BAB^ABA

  • 定义：所有从 AAA 到 BBB 的函数构成的集合。
  • 记作：BA={f∣f:A→B}B^A = \{ f \mid f: A \to B \}BA={f∣f:A→B}。
  • 基数（数量）：如果 ∣A∣=m,∣B∣=n|A|=m, |B|=n∣A∣=m,∣B∣=n，且 m,n>0m, n > 0m,n>0，则 ∣BA∣=nm|B^A| = n^m∣BA∣=nm。
    • 记忆：每个 AAA 中的元素都有 nnn 种选择，共有 mmm 个元素，所以是 n×n×⋯×nn \times n \times \dots \times nn×n×⋯×n (mmm 次)。

5. 像与完全原像

设 f:A→Bf: A \to Bf:A→B，且 A1⊆A,B1⊆BA_1 \subseteq A, B_1 \subseteq BA1​⊆A,B1​⊆B。

  • 像 (Image)：集合 A1A_1A1​ 中所有元素的函数值组成的集合。

    f(A1)={f(x)∣x∈A1}f(A_1) = \{ f(x) \mid x \in A_1 \} f(A1​)={f(x)∣x∈A1​}

    • 特例：f(A)f(A)f(A) 就是函数的值域。
  • 完全原像 (Inverse Image)：集合 B1B_1B1​ 中所有元素的原像组成的集合。

    f−1(B1)={x∣x∈A∧f(x)∈B1}f^{-1}(B_1) = \{ x \mid x \in A \land f(x) \in B_1 \} f−1(B1​)={x∣x∈A∧f(x)∈B1​}

    • 注意：这里 f−1f^{-1}f−1 并不是反函数，而是表示一种集合运算。即使 fff 不可逆，完全原像也是存在的。

6. 函数的类型（满射、单射、双射）

这是函数最重要的分类性质。

类型 名称 定义 直观理解
满射 Surjection / Onto ranf=B\text{ran}f = Branf=B 陪域无剩余。
B 中的每个元素至少被 A 中的一个元素射中。
单射 Injection / One-to-One ∀x1,x2∈A,f(x1)=f(x2)⇒x1=x2\forall x_1, x_2 \in A, f(x_1)=f(x_2) \Rightarrow x_1=x_2∀x1​,x2​∈A,f(x1​)=f(x2​)⇒x1​=x2​ 不撞车。
不同的输入一定对应不同的输出。
双射 Bijection 既是满射又是单射 一一对应。
A 和 B 的元素完美配对，不多不少。

7. 特殊函数

  • 常函数：对于所有 x∈Ax \in Ax∈A，都有 f(x)=cf(x) = cf(x)=c（ccc 是固定值）。
  • 恒等函数：IA:A→AI_A: A \to AIA​:A→A，对所有 x∈Ax \in Ax∈A，有 IA(x)=xI_A(x) = xIA​(x)=x。
  • 特征函数：设 AAA 是全集 EEE 的子集，χA:E→{0,1}\chi_A: E \to \{0, 1\}χA​:E→{0,1}。
    • 若 x∈Ax \in Ax∈A，χA(x)=1\chi_A(x) = 1χA​(x)=1。
    • 若 x∉Ax \notin Ax∈/A，χA(x)=0\chi_A(x) = 0χA​(x)=0。
  • 自然映射：设 RRR 是 AAA 上的等价关系，g:A→A/Rg: A \to A/Rg:A→A/R 定义为 g(a)=[a]Rg(a) = [a]_Rg(a)=[a]R​（把元素映射到它所属的等价类）。自然映射通常是满射。

8.2 函数的复合与反函数

1. 函数的复合 (Composition)

  • 定义：设 F,GF, GF,G 是函数，它们的复合 F∘GF \circ GF∘G 也是函数。
  • 顺序说明（重要）：
    本书沿用二元关系的复合定义（左复合），即 F∘GF \circ GF∘G 表示先进行 FFF，后进行 GGG。
  • 公式：

    (F∘G)(x)=G(F(x))(F \circ G)(x) = G(F(x)) (F∘G)(x)=G(F(x))

    • 注意：这与微积分中常用的 (g∘f)(x)=g(f(x))(g \circ f)(x) = g(f(x))(g∘f)(x)=g(f(x)) 符号顺序相反，请务必根据 FFF 和 GGG 的定义域和值域来判断顺序。
  • 存在条件：要使 f∘gf \circ gf∘g 有意义，必须满足 ranf⊆domg\text{ran}f \subseteq \text{dom}granf⊆domg（fff 的输出必须在 ggg 的输入范围内）。
  • 性质：
    • 结合律：(f∘g)∘h=f∘(g∘h)(f \circ g) \circ h = f \circ (g \circ h)(f∘g)∘h=f∘(g∘h)。
    • 保性：
      • 两个满射的复合是满射。
      • 两个单射的复合是单射。
      • 两个双射的复合是双射。

2. 反函数 (Inverse Function)

  • 定义：设 f:A→Bf: A \to Bf:A→B 是函数，其逆关系 f−1f^{-1}f−1 称为 fff 的反函数。
  • 存在条件：
    • f−1f^{-1}f−1 是函数的充要条件是：fff 是双射（既是单射又是满射）。
    • 如果 fff 只是单射但不是满射，f−1f^{-1}f−1 只是从 ranf\text{ran}franf 到 AAA 的函数，而不是从 BBB 到 AAA 的函数。
  • 性质：
    • f−1∘f=IAf^{-1} \circ f = I_Af−1∘f=IA​ （先过去再回来，回到原点）。
    • f∘f−1=IBf \circ f^{-1} = I_Bf∘f−1=IB​。
    • (f−1)−1=f(f^{-1})^{-1} = f(f−1)−1=f。

8.3 双射函数与集合的基数

本节利用双射来比较集合的大小（基数），特别是无穷集合的大小。

1. 集合的等势 (Equipotence)

  • 定义：如果集合 AAA 和 BBB 之间存在一个双射函数，则称 AAA 与 BBB 等势（或基数相同）。
  • 记作：A≈BA \approx BA≈B。
  • 性质：等势关系是等价关系（满足自反性、对称性、传递性）。
  • 重要实例：
    • N≈Z\mathbb{N} \approx \mathbb{Z}N≈Z （自然数与整数一样多）。
    • N≈Q\mathbb{N} \approx \mathbb{Q}N≈Q （自然数与有理数一样多）。
    • N×N≈N\mathbb{N} \times \mathbb{N} \approx \mathbb{N}N×N≈N。
    • (0,1)≈R(0, 1) \approx \mathbb{R}(0,1)≈R （开区间与全体实数一样多）。

2. 优势与劣势 (Dominance)

  • 定义：
    • 如果存在从 AAA 到 BBB 的单射函数，则称 AAA 的势小于等于 BBB 的势（AAA 劣势于 BBB）。记作 A≼BA \preccurlyeq BA≼B。
    • 如果 A≼BA \preccurlyeq BA≼B 且 A≉BA \not\approx BA≈B（即不存在双射），则称 AAA 的势严格小于 BBB 的势。记作 A≺BA \prec BA≺B。

3. 两个重要定理

  1. 康托尔定理 (Cantor’s Theorem)：
    对于任意集合 AAA，都有 A≺P(A)A \prec P(A)A≺P(A)。

    • 含义：集合的幂集（所有子集的集合）的基数总是严格大于原集合的基数。这意味着不存在最大的集合。

  2. 康托尔-施罗德-伯恩斯坦定理 (Cantor-Schroder-Bernstein Theorem)：
    设 A,BA, BA,B 是集合，如果 A≼BA \preccurlyeq BA≼B 且 B≼AB \preccurlyeq AB≼A，则 A≈BA \approx BA≈B。

    • 作用：要证明两个集合一样大，只需分别找到彼此的一个单射即可，不需要直接构造复杂的双射。

4. 基数 (Cardinal Numbers)

用来度量集合“大小”的数。记作 cardA\text{card}AcardA 或 ∣A∣|A|∣A∣。

  • 有穷集：如果 A≈{1,2,…,n}A \approx \{1, 2, \dots, n\}A≈{1,2,…,n}，则 cardA=n\text{card}A = ncardA=n。

  • 无穷集：

    • 可数集 (Countable)：与自然数集 N\mathbb{N}N 等势的集合。
      • 基数记作 ℵ0\aleph_0ℵ0​ （读作阿列夫零）。
      • 例子：Z,Q\mathbb{Z}, \mathbb{Q}Z,Q 都是 ℵ0\aleph_0ℵ0​。
    • 不可数集 (Uncountable)：比 N\mathbb{N}N 势大的集合。
      • 连续统势：与实数集 R\mathbb{R}R 等势的集合。
      • 基数记作 ℵ\alephℵ 或 ℵ1\aleph_1ℵ1​（在连续统假设下）。
      • 例子：(0,1),C,P(N)(0, 1), \mathbb{C}, P(\mathbb{N})(0,1),C,P(N) 都是不可数集。

  • 基数的运算关系：

    • n<ℵ0<ℵn < \aleph_0 < \alephn<ℵ0​<ℵ。
    • 2ℵ0=ℵ2^{\aleph_0} = \aleph2ℵ0​=ℵ （自然数集的幂集势等于实数集的势）。
