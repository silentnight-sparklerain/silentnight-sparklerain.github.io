第12章 基本的组合计数公式

12.1 加法法则与乘法法则

这是组合计数中最基本的两个原则。

1. 加法法则 (Addition Principle)

  • 适用场景：分类计数（"或"的情况）。
  • 定义：设事件 AAA 有 mmm 种产生方式，事件 BBB 有 nnn 种产生方式。如果 AAA 与 BBB 的产生方式不重叠（互斥），那么“事件 AAA 或 BBB”共有 m+nm+nm+n 种产生方式。
  • 集合语言：若 A∩B=∅A \cap B = \varnothingA∩B=∅，则 ∣A∪B∣=∣A∣+∣B∣|A \cup B| = |A| + |B|∣A∪B∣=∣A∣+∣B∣。
  • 推广：若 nnn 个事件 A1,A2,…,AnA_1, A_2, \dots, A_nA1​,A2​,…,An​ 两两互斥，且分别有 p1,p2,…,pnp_1, p_2, \dots, p_np1​,p2​,…,pn​ 种方式，则从中任选一个事件发生共有 p1+p2+⋯+pnp_1 + p_2 + \dots + p_np1​+p2​+⋯+pn​ 种方式。

2. 乘法法则 (Multiplication Principle)

  • 适用场景：分步计数（"且"的情况）。
  • 定义：设事件 AAA 有 mmm 种产生方式，事件 BBB 有 nnn 种产生方式。如果 AAA 与 BBB 的产生方式彼此独立（互不影响），那么“事件 AAA 与 BBB”共有 m×nm \times nm×n 种产生方式。
  • 集合语言：∣A×B∣=∣A∣⋅∣B∣|A \times B| = |A| \cdot |B|∣A×B∣=∣A∣⋅∣B∣。
  • 推广：若一个过程分为 nnn 个步骤，第 iii 步有 pip_ipi​ 种选择，且每一步的选择都独立于之前的选择，则完成整个过程共有 p1×p2×⋯×pnp_1 \times p_2 \times \dots \times p_np1​×p2​×⋯×pn​ 种方式。

12.2 排列与组合

排列和组合的计数是基本的计数问题。其核心在于两个维度：

  1. 有序 vs 无序：选出来的元素是否讲究顺序？
  2. 不重复 vs 可重复：同一个元素是否可以被多次选取？

1. 集合的排列 (Permutation of Sets) —— 无重复，有序

这是最经典的模型：从一堆不同的东西里，选出几个来排队。

  • 排列：
    设 SSS 为 nnn 元集（包含 nnn 个不同元素的集合）。
    • rrr 排列：从 SSS 中有序选取的 rrr 个元素称为 SSS 的一个 rrr 排列。
    • 全排列：当 r=nr=nr=n 时，称为 SSS 的全排列。
  • 记号：
    SSS 的不同 rrr 排列总数记作 P(n,r)P(n, r)P(n,r)。
  • 排列数公式：

    P(n,r)=n(n−1)(n−2)…(n−r+1)=n!(n−r)!P(n, r) = n(n-1)(n-2)\dots(n-r+1) = \frac{n!}{(n-r)!} P(n,r)=n(n−1)(n−2)…(n−r+1)=(n−r)!n!​

    • 解释：
      • 第 1 个位置有 nnn 种选择；
      • 第 2 个位置剩 n−1n-1n−1 种选择；
      • ……
      • 第 rrr 个位置剩 n−r+1n-r+1n−r+1 种选择。
    • 规定：0!=10! = 10!=1。
    • 全排列：P(n,n)=n!P(n, n) = n!P(n,n)=n!。

2. 集合的组合 (Combination of Sets) —— 无重复，无序

这是另一个经典模型：从一堆不同的东西里，选出几个来组成一组（不排队）。

  • 组合：
    从 SSS 中无序选取的 rrr 个元素称为 SSS 的一个 rrr 组合。
    • 本质：SSS 的一个 rrr 组合就是 SSS 的一个基数为 rrr 的子集。
  • 记号：
    SSS 的不同 rrr 组合总数记作 C(n,r)C(n, r)C(n,r)，或者更通用的二项式系数符号 (nr)\binom{n}{r}(rn​)。
  • 组合数公式：

    C(n,r)=(nr)=P(n,r)r!=n!r!(n−r)!C(n, r) = \binom{n}{r} = \frac{P(n, r)}{r!} = \frac{n!}{r!(n-r)!} C(n,r)=(rn​)=r!P(n,r)​=r!(n−r)!n!​

    • 解释：
      先按有序选出 rrr 个元素（P(n,r)P(n, r)P(n,r) 种），但因为我们不在乎顺序，所以要把这 rrr 个元素内部的排列数（r!r!r! 种）除掉。
  • 重要性质：
    1. 对称性：C(n,r)=C(n,n−r)C(n, r) = C(n, n-r)C(n,r)=C(n,n−r)。
      • 解释：选 rrr 个带走，等于选 n−rn-rn−r 个留下。
    2. 杨辉三角递推式：C(n,r)=C(n−1,r−1)+C(n−1,r)C(n, r) = C(n-1, r-1) + C(n-1, r)C(n,r)=C(n−1,r−1)+C(n−1,r)。
      • 解释：对于某个特定元素 xxx，选出的 rrr 个里要么包含 xxx（剩下需从 n−1n-1n−1 里选 r−1r-1r−1），要么不包含 xxx（需从 n−1n-1n−1 里选 rrr）。

3. 环排列 (Circular Permutation)

  • 定义：
    nnn 个不同元素围成一个圆圈的排列。
    • 特点：旋转后重合的排列视为同一种排列（没有固定的“起点”）。
  • 推论：
    nnn 个不同元素的环排列总数为：

    P(n,n)n=(n−1)!\frac{P(n, n)}{n} = (n-1)! nP(n,n)​=(n−1)!

    • 解释：线排列有 n!n!n! 种，围成圈后，每 nnn 种旋转状态对应同一个环排列，所以除以 nnn。
    • 更通用的理解：固定其中一个人坐下（作为参照物），剩下 n−1n-1n−1 个人随便排。

4. 多重集的排列 (Permutation of Multisets) —— 有重复，有序

如果不让选重复元素，但集合里本身就有重复元素（比如 “BOOK” 里有两个 ‘O’），怎么排？

  • 多重集定义：
    设 S={n1⋅a1,n2⋅a2,…,nk⋅ak}S = \{n_1 \cdot a_1, n_2 \cdot a_2, \dots, n_k \cdot a_k\}S={n1​⋅a1​,n2​⋅a2​,…,nk​⋅ak​} 为多重集。
    • a1,…,aka_1, \dots, a_ka1​,…,ak​ 是 kkk 种不同的元素。
    • nin_ini​ 是元素 aia_iai​ 的重复度（即有多少个 aia_iai​）。
    • n=n1+⋯+nkn = n_1 + \dots + n_kn=n1​+⋯+nk​ 是 SSS 中元素的总数。
    • ni=∞n_i = \inftyni​=∞ 表示元素 aia_iai​ 有无限多个（足够多）。
  • 有限多重集的全排列：
    设 SSS 的元素总数为 nnn，且每个元素的重复度有限。则 SSS 的全排列数为：

    n!n1!n2!…nk!\frac{n!}{n_1! n_2! \dots n_k!} n1​!n2​!…nk​!n!​

    • 解释：先假设所有元素都不同，有 n!n!n! 种排法。然后因为 n1n_1n1​ 个 a1a_1a1​ 是相同的，它们内部交换位置看不出来，所以要除以 n1!n_1!n1​!；同理除以 n2!,…,nk!n_2!, \dots, n_k!n2​!,…,nk​!。
    • 这个数也记作多项式系数 (nn1,n2,…,nk)\binom{n}{n_1, n_2, \dots, n_k}(n1​,n2​,…,nk​n​)。
  • 推论 (无限多重集的 rrr 排列)：
    如果每种元素的重复度都是 ∞\infty∞（或者至少 ≥r\ge r≥r），那么从 kkk 种元素中选取 rrr 个进行排列的方法数为：

    krk^r kr

    • 解释：每个位置都有 kkk 种选择，互不影响。

5. 多重集的组合 (Combination of Multisets) —— 有重复，无序

这是最容易混淆的模型：允许重复选取元素，且不考虑顺序。

  • 定义：
    从多重集 SSS 中选取 rrr 个元素作为子多重集。如果 SSS 中每种元素的重复度都至少为 rrr（即 ∞⋅a1,…,∞⋅ak\infty \cdot a_1, \dots, \infty \cdot a_k∞⋅a1​,…,∞⋅ak​），这被称为允许重复的组合。
  • 等价模型：
    1. 从 kkk 类不同物品中，取 rrr 个物品（允许重复取）。
    2. 将 rrr 个相同的小球，放入 kkk 个不同的盒子中（允许有空盒）。
    3. 求不定方程 x1+x2+⋯+xk=rx_1 + x_2 + \dots + x_k = rx1​+x2​+⋯+xk​=r 的非负整数解的个数。
  • 隔板法：
    从 kkk 个不同元素中，允许重复地选取 rrr 个元素的组合数是：

    C(r+k−1,r)=C(r+k−1,k−1)C(r+k-1, r) = C(r+k-1, k-1) C(r+k−1,r)=C(r+k−1,k−1)

    • 证明思路（隔板法）：
      把 rrr 个小球排成一排，在中间插入 k−1k-1k−1 个“隔板”。
      隔板把球分成了 kkk 部分，第 iii 部分的球数就是第 iii 类元素的选取数量。
      总位置数 = 球数 rrr + 隔板数 k−1k-1k−1。
      我们需要从中选出 k−1k-1k−1 个位置放隔板（或者选 rrr 个位置放球）。
  • 注意：
    这个公式要求每种元素都足够多（至少 rrr 个）。如果某些元素的数量有限制（比如第1种元素最多只能取2个），则不能直接用此公式，通常需要用生成函数或容斥原理求解。

12.3 二项式定理与组合恒等式

1. 二项式定理 (Binomial Theorem)

  • 二项式定理：
    设 nnn 是正整数，x,yx, yx,y 为任意实数（或复数），则：

    (x+y)n=∑k=0n(nk)xkyn−k(x+y)^n = \sum_{k=0}^n \binom{n}{k} x^k y^{n-k} (x+y)n=k=0∑n​(kn​)xkyn−k

  • 二项式系数：
    展开式中 xkyn−kx^k y^{n-k}xkyn−k 的系数 (nk)\binom{n}{k}(kn​)（即 C(n,k)C(n, k)C(n,k)）称为二项式系数。
  • 组合意义：
    在 (x+y)(x+y)…(x+y)(x+y)(x+y)\dots(x+y)(x+y)(x+y)…(x+y) 的 nnn 个括号中，每一项的乘积是通过从每个括号里选 xxx 或选 yyy 得到的。
    如果要得到 xkyn−kx^k y^{n-k}xkyn−k，必须恰好在 kkk 个括号中选 xxx，在剩下的 n−kn-kn−k 个括号中选 yyy。从 nnn 个括号中选 kkk 个选 xxx 的方法数正是 (nk)\binom{n}{k}(kn​)。

2. 基本组合恒等式

基于二项式定理，通过对 x,yx, yx,y 赋予特定的值，可以推导出许多重要的恒等式。

  1. 对称性：

    (nk)=(nn−k)\binom{n}{k} = \binom{n}{n-k} (kn​)=(n−kn​)

    • 意义：选 kkk 个带走等于选 n−kn-kn−k 个留下。
  2. 帕斯卡公式 (Pascal’s Identity) / 递推性：

    (nk)=(n−1k)+(n−1k−1)\binom{n}{k} = \binom{n-1}{k} + \binom{n-1}{k-1} (kn​)=(kn−1​)+(k−1n−1​)

    • 意义：组合数的递推关系，对应杨辉三角。
  3. 吸收/提取公式：

    (nk)=nk(n−1k−1)(k≥1)\binom{n}{k} = \frac{n}{k} \binom{n-1}{k-1} \quad (k \ge 1) (kn​)=kn​(k−1n−1​)(k≥1)

    k(nk)=n(n−1k−1)k \binom{n}{k} = n \binom{n-1}{k-1} k(kn​)=n(k−1n−1​)

    • 作用：常用于将组合数中的变量 kkk 移到外面，或者降低组合数的阶数。
  4. 系数之和：
    令 x=1,y=1x=1, y=1x=1,y=1，代入二项式定理：

    ∑k=0n(nk)=2n\sum_{k=0}^n \binom{n}{k} = 2^n k=0∑n​(kn​)=2n

    • 意义：nnn 元集的所有子集个数为 2n2^n2n。
  5. 交错和：
    令 x=−1,y=1x=-1, y=1x=−1,y=1，代入二项式定理：

    ∑k=0n(−1)k(nk)=0\sum_{k=0}^n (-1)^k \binom{n}{k} = 0 k=0∑n​(−1)k(kn​)=0

    • 意义：奇数个元素的子集个数等于偶数个元素的子集个数（即 ∑(n2k)=∑(n2k+1)=2n−1\sum \binom{n}{2k} = \sum \binom{n}{2k+1} = 2^{n-1}∑(2kn​)=∑(2k+1n​)=2n−1）。

3. 范德蒙德恒等式及其推广

这类恒等式涉及两个或多个集合的组合计数，通常采用组合分析法（算两次）证明。

  • 积的组合恒等式 (Subset of a Subset)：

    (nk)(kr)=(nr)(n−rk−r)\binom{n}{k} \binom{k}{r} = \binom{n}{r} \binom{n-r}{k-r} (kn​)(rk​)=(rn​)(k−rn−r​)

    • 组合意义：
      • 左边：先从 nnn 人中选 kkk 人组成“代表团”，再从 kkk 人中选 rrr 人当“核心成员”。
      • 右边：先从 nnn 人中直接选 rrr 人当“核心成员”，再从剩下 n−rn-rn−r 人中选 k−rk-rk−r 人补充进“代表团”。
  • 范德蒙德恒等式 (Vandermonde’s Identity)：

    ∑k=0r(mk)(nr−k)=(m+nr)\sum_{k=0}^r \binom{m}{k} \binom{n}{r-k} = \binom{m+n}{r} k=0∑r​(km​)(r−kn​)=(rm+n​)

    • 条件：r≤min⁡(m,n)r \le \min(m, n)r≤min(m,n)。
    • 组合意义：
      设甲组有 mmm 人，乙组有 nnn 人，共 m+nm+nm+n 人。从中选出 rrr 人的方法数 (m+nr)\binom{m+n}{r}(rm+n​)。
      这等价于：枚举从甲组选 kkk 人，从乙组选 r−kr-kr−k 人，并将所有可能的 kkk 值（000 到 rrr）的情况相加。
  • 平方和恒等式：
    这是范德蒙德恒等式的特例（令 m=n,r=nm=n, r=nm=n,r=n，并利用对称性 (nn−k)=(nk)\binom{n}{n-k} = \binom{n}{k}(n−kn​)=(kn​)）：

    ∑k=0n(nk)2=(2nn)\sum_{k=0}^n \binom{n}{k}^2 = \binom{2n}{n} k=0∑n​(kn​)2=(n2n​)

4. 涉及导数的恒等式

利用微积分方法处理二项式定理，可以求出带系数 kkk 的组合数和。

  • 一阶矩：

    ∑k=0nk(nk)=n2n−1\sum_{k=0}^n k \binom{n}{k} = n 2^{n-1} k=0∑n​k(kn​)=n2n−1

5. 多项式定理 (Multinomial Theorem)

这是二项式定理在多项展开上的推广。

  • 多项式定理：
    设 nnn 为正整数，则 (x1+x2+⋯+xt)n(x_1 + x_2 + \dots + x_t)^n(x1​+x2​+⋯+xt​)n 的展开式为：

    ∑n!n1!n2!…nt!x1n1x2n2…xtnt\sum \frac{n!}{n_1! n_2! \dots n_t!} x_1^{n_1} x_2^{n_2} \dots x_t^{n_t} ∑n1​!n2​!…nt​!n!​x1n1​​x2n2​​…xtnt​​

    • 求和条件：对所有满足 n1+n2+⋯+nt=nn_1 + n_2 + \dots + n_t = nn1​+n2​+⋯+nt​=n 的非负整数解组 (n1,…,nt)(n_1, \dots, n_t)(n1​,…,nt​) 求和。
  • 多项式系数：
    系数 n!n1!n2!…nt!\frac{n!}{n_1! n_2! \dots n_t!}n1​!n2​!…nt​!n!​ 记作 (nn1n2…nt)\binom{n}{n_1 n_2 \dots n_t}(n1​n2​…nt​n​)，称为多项式系数。
    • 它恰好等于多重集 {n1⋅x1,…,nt⋅xt}\{ n_1 \cdot x_1, \dots, n_t \cdot x_t \}{n1​⋅x1​,…,nt​⋅xt​} 的全排列数。

    展开式中不同项的个数等于方程 n1+⋯+nt=nn_1 + \dots + n_t = nn1​+⋯+nt​=n 的非负整数解的个数，即 (n+t−1n)\binom{n+t-1}{n}(nn+t−1​)（隔板法）。

12.4 非降路径问题 (Catalan Number 相关)

1. 格路计数

  • 问题：在二维网格中，从 (0,0)(0,0)(0,0) 走到 (m,n)(m, n)(m,n)，每步只能向右 (x+1x+1x+1) 或向上 (y+1y+1y+1)，共有多少种路径？
  • 解法：总共需要走 m+nm+nm+n 步，其中必须有 mmm 步向右，nnn 步向上。
  • 公式：

    (m+nm)=(m+nn)\binom{m+n}{m} = \binom{m+n}{n} (mm+n​)=(nm+n​)

2. 卡特兰数 (Catalan Number)

  • 问题：从 (0,0)(0,0)(0,0) 到 (n,n)(n,n)(n,n) 的路径中，不越过对角线 y=xy=xy=x（即始终满足 y≤xy \le xy≤x 或始终 y≥xy \ge xy≥x）的路径数。
  • 公式：

    Cn=1n+1(2nn)C_n = \frac{1}{n+1} \binom{2n}{n} Cn​=n+11​(n2n​)

  • 应用：
    • nnn 对括号的合法匹配数。
    • 凸 n+2n+2n+2 边形的三角剖分数。
    • nnn 个节点的二叉树形态数。
    • 进出栈序列数。

13.1 递推方程的定义及实例

1. 递推方程 (Recurrence Relation)

  • 定义：设序列 a0,a1,…,an,…a_0, a_1, \dots, a_n, \dotsa0​,a1​,…,an​,…。一个把 ana_nan​ 与某些个 ai(i<n)a_i (i < n)ai​(i<n) 联系起来的等式，称为关于序列 {an}\{a_n\}{an​} 的递推方程。

    • 相当于：nnn 个不同球放入 mmm 个相同盒子，无空盒。
  • 记号：S(n,m)S(n, m)S(n,m) 或 {nm}\left\{ \begin{smallmatrix} n \\ m \end{smallmatrix} \right\}{nm​}。
  • 递推公式：

    S(n,m)=S(n−1,m−1)+m⋅S(n−1,m)S(n, m) = S(n-1, m-1) + m \cdot S(n-1, m) S(n,m)=S(n−1,m−1)+m⋅S(n−1,m)

    • 解释：对于第 nnn 个元素，要么它自己单独一组（S(n−1,m−1)S(n-1, m-1)S(n−1,m−1)），要么放入已有的 mmm 个组中的某一个（m⋅S(n−1,m)m \cdot S(n-1, m)m⋅S(n−1,m)）。
  • 性质：
    • S(n,1)=1,S(n,n)=1S(n, 1) = 1, S(n, n) = 1S(n,1)=1,S(n,n)=1。
    • xn=∑k=0nS(n,k)[x]kx^n = \sum_{k=0}^n S(n, k) [x]_kxn=∑k=0n​S(n,k)[x]k​，其中 [x]k=x(x−1)…(x−k+1)[x]_k = x(x-1)\dots(x-k+1)[x]k​=x(x−1)…(x−k+1) 是下降阶乘幂。

(2) 第一类 Stirling 数 s(n,m)s(n, m)s(n,m)

  • 定义：将 nnn 个不同元素排成 mmm 个非空轮换（Cycle）的方法数。
  • 记号：s(n,m)s(n, m)s(n,m) 或 [nm]\left[ \begin{smallmatrix} n \\ m \end{smallmatrix} \right][nm​]。
  • 递推公式：

    s(n,m)=s(n−1,m−1)+(n−1)s(n−1,m)s(n, m) = s(n-1, m-1) + (n-1)s(n-1, m) s(n,m)=s(n−1,m−1)+(n−1)s(n−1,m)

    • 解释：第 nnn 个元素要么自成一环，要么插入到前 n−1n-1n−1 个元素组成的 mmm 个环中的任意一个元素的左边（共 n−1n-1n−1 个位置）。
