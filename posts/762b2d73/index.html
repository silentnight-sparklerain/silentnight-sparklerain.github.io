第19章 初等数论

19.1 素数

1. 整除性 (Divisibility)

  • 定义：设 a,ba, ba,b 是两个整数，且 b≠0b \neq 0b=0。如果存在整数 ccc 使得 a=bca = bca=bc，则称 aaa 被 bbb 整除，或 bbb 整除 aaa。
  • 记作：b∣ab \mid ab∣a。
    • 若 bbb 不能整除 aaa，记作 b∤ab \nmid ab∤a。
  • 术语：
    • aaa 称为 bbb 的倍数 (Multiple)。
    • bbb 称为 aaa 的因子 (Factor) 或约数。
  • 因子分类：
    • 平凡因子：111 和它自己。
    • 真因子：除平凡因子之外的因子。
  • 性质：
    1. b∣a  ⟺  a mod b=0b \mid a \iff a \bmod b = 0b∣a⟺amodb=0。
    2. 传递性：若 a∣ba \mid ba∣b 且 b∣cb \mid cb∣c，则 a∣ca \mid ca∣c。
    3. 线性组合：若 a∣ba \mid ba∣b 且 a∣ca \mid ca∣c，则对于任意整数 x,yx, yx,y，有 a∣(xb+yc)a \mid (xb + yc)a∣(xb+yc)。
    4. 若 a∣ba \mid ba∣b 且 b∣ab \mid ab∣a，则 a=±ba = \pm ba=±b。
    5. 若 a∣ba \mid ba∣b 且 b≠0b \neq 0b=0，则 ∣a∣≤∣b∣|a| \le |b|∣a∣≤∣b∣。

2. 带余除法 (Division Algorithm)

  • 定理：设 a,ba, ba,b 是两个整数，且 b≠0b \neq 0b=0，则存在唯一的整数 qqq (商) 和 rrr (余数)，使得：

    a=qb+r,0≤r<∣b∣a = qb + r, \quad 0 \le r < |b| a=qb+r,0≤r<∣b∣

  • 记号：r=a mod br = a \bmod br=amodb。

3. 素数与合数

  • 素数 (Prime Number)：如果正整数 a>1a > 1a>1 且只能被 1 和它自己整除。
  • 合数 (Composite Number)：如果正整数 a>1a > 1a>1 且不是素数（即除了 1 和自身外还有其他因子）。
  • 基石性质：
    • 如果 ppp 是素数且 p∣abp \mid abp∣ab，则必然有 p∣ap \mid ap∣a 或者 p∣bp \mid bp∣b。
    • 算术基本定理 (Fundamental Theorem of Arithmetic)：任何大于 1 的整数 aaa 都可以唯一地分解为素数的乘积（不计顺序）。

      a=p1r1p2r2…pkrka = p_1^{r_1} p_2^{r_2} \dots p_k^{r_k} a=p1r1​​p2r2​​…pkrk​​

      这称为 aaa 的素因子分解或标准分解式。

4. 素数的分布

  • 素数有无穷多个：不存在最大的素数（欧几里得证明）。
  • 素数定理：π(n)\pi(n)π(n) 表示小于等于 nnn 的素数个数。当 nnn 很大时，π(n)\pi(n)π(n) 近似等于 nln⁡n\frac{n}{\ln n}lnnn​。

    lim⁡n→∞π(n)n/ln⁡n=1\lim_{n \to \infty} \frac{\pi(n)}{n / \ln n} = 1 n→∞lim​n/lnnπ(n)​=1

  • 判定范围定理：如果 aaa 是一个合数，则 aaa 必有一个小于等于 a\sqrt{a}a​ 的真因子。
    • 应用：要判断 aaa 是否为素数，只需检验 222 到 a\sqrt{a}a​ 之间的素数是否能整除 aaa。
  • 埃拉托斯特尼筛法 (Sieve of Eratosthenes)：一种找出一定范围内所有素数的算法（挖掉所有素数的倍数）。

5. 梅森素数 (Mersenne Primes)

  • 形如 2n−12^n - 12n−1 的素数。
  • 必要条件：若 2n−12^n - 12n−1 是素数，则 nnn 必须是素数（但反之不一定成立）。

19.2 最大公约数与最小公倍数

1. 定义

  • 公因子：既整除 aaa 又整除 bbb 的数。
  • 最大公约数 (GCD)：所有公因子中最大的那个。记作 gcd⁡(a,b)\gcd(a, b)gcd(a,b) 或简写为 (a,b)(a, b)(a,b)。
  • 公倍数：既是 aaa 的倍数又是 bbb 的倍数的数。
  • 最小公倍数 (LCM)：所有正公倍数中最小的那个。记作 lcm(a,b)\text{lcm}(a, b)lcm(a,b)。
  • 关系：gcd⁡(a,b)×lcm(a,b)=∣ab∣\gcd(a, b) \times \text{lcm}(a, b) = |ab|gcd(a,b)×lcm(a,b)=∣ab∣。

2. 欧几里得算法 (Euclidean Algorithm) / 辗转相除法

  • 核心原理：gcd⁡(a,b)=gcd⁡(b,a mod b)\gcd(a, b) = \gcd(b, a \bmod b)gcd(a,b)=gcd(b,amodb)。
  • 过程：通过反复做带余除法，直到余数为 0，最后的除数就是最大公约数。

    rk−2=qkrk−1+rk(最后rk≠0,rk+1=0⇒gcd⁡=rk)r_{k-2} = q_k r_{k-1} + r_k \quad (\text{最后} r_k \neq 0, r_{k+1}=0 \Rightarrow \gcd = r_k) rk−2​=qk​rk−1​+rk​(最后rk​=0,rk+1​=0⇒gcd=rk​)

3. 扩展欧几里得算法与贝祖等式

  • 定理 19.7：设 a,ba, ba,b 不全为 0，则存在整数 x,yx, yx,y 使得：

    gcd⁡(a,b)=xa+yb\gcd(a, b) = xa + yb gcd(a,b)=xa+yb

  • 算法：在做辗转相除法的过程中，通过回代，可以求出系数 xxx 和 yyy。

4. 互素 (Coprime / Relatively Prime)

  • 定义：如果 gcd⁡(a,b)=1\gcd(a, b) = 1gcd(a,b)=1，则称 aaa 和 bbb 互素。
  • 充要条件：a,ba, ba,b 互素   ⟺  \iff⟺ 存在整数 x,yx, yx,y 使得 ax+by=1ax + by = 1ax+by=1。
  • 重要性质：如果 a∣bca \mid bca∣bc 且 gcd⁡(a,b)=1\gcd(a, b) = 1gcd(a,b)=1，则 a∣ca \mid ca∣c。

19.3 同余 (Congruence)

1. 基本概念

  • 模 (Modulus)：设 mmm 是正整数。
  • 同余定义：如果 m∣(a−b)m \mid (a - b)m∣(a−b)（即 aaa 和 bbb 除以 mmm 的余数相同），则称 aaa 与 bbb 模 mmm 同余。
  • 记作：a≡b(modm)a \equiv b \pmod ma≡b(modm)。
  • 不同余：记作 a≢b(modm)a \not\equiv b \pmod ma≡b(modm)。

2. 同余的性质

  • 等价关系：同余关系具有自反性、对称性、传递性。
  • 保运算性：如果 a≡b(modm)a \equiv b \pmod ma≡b(modm)，c≡d(modm)c \equiv d \pmod mc≡d(modm)，则：
    1. a±c≡b±d(modm)a \pm c \equiv b \pm d \pmod ma±c≡b±d(modm)
    2. ac≡bd(modm)ac \equiv bd \pmod mac≡bd(modm)
    3. ak≡bk(modm)a^k \equiv b^k \pmod mak≡bk(modm) (kkk 为非负整数)
    • 注意：除法不一定保持同余（除非除数与模互素）。

3. 剩余类 (Residue Classes)

  • Zm\mathbb{Z}_mZm​：模 mmm 的剩余类集合，包含 {0,1,…,m−1}\{0, 1, \dots, m-1\}{0,1,…,m−1}。
  • 在 Zm\mathbb{Z}_mZm​ 上的加法和乘法都是模 mmm 进行的。

4. 应用实例

  • 大数求余：利用同余的乘法和幂运算性质，将大数的运算转化为小数的运算。
  • 日期计算：利用模 7 的同余性质计算星期几（公历的周期性）。

19.4 一次同余方程

1. 定义

形如 ax≡c(modm)ax \equiv c \pmod max≡c(modm) 的方程，其中 xxx 是未知数。

2. 解的判定 (定理 19.9)

设 d=gcd⁡(a,m)d = \gcd(a, m)d=gcd(a,m)。

  • 方程有解的充要条件是：d∣cd \mid cd∣c （即 ccc 是 aaa 和 mmm 最大公约数的倍数）。
  • 如果有解，则在模 mmm 下恰有 ddd 个不通的解。
  • 特别地：如果 gcd⁡(a,m)=1\gcd(a, m) = 1gcd(a,m)=1，则方程有唯一解。

3. 模逆 (Modular Inverse)

  • 定义：如果 ab≡1(modm)ab \equiv 1 \pmod mab≡1(modm)，则称 bbb 是 aaa 的模 mmm 逆。记作 a−1(modm)a^{-1} \pmod ma−1(modm)。
    • 通俗理解：同余世界里的“倒数”。
  • 存在条件：aaa 存在模 mmm 逆   ⟺  gcd⁡(a,m)=1\iff \gcd(a, m) = 1⟺gcd(a,m)=1（a,ma, ma,m 互素）。
  • 求逆方法：
    1. 扩展欧几里得法：求解 ax+my=1ax + my = 1ax+my=1，则 xxx 就是 a−1a^{-1}a−1。
    2. 欧拉定理/费马小定理（见下节）。

4. 解方程的方法

要解 ax≡c(modm)ax \equiv c \pmod max≡c(modm)：

  1. 先求 aaa 的逆 a−1a^{-1}a−1。
  2. 两边同乘 a−1a^{-1}a−1，得到 x≡a−1c(modm)x \equiv a^{-1}c \pmod mx≡a−1c(modm)。

19.5 欧拉定理和费马小定理

1. 欧拉函数 (Euler’s Totient Function)

  • 定义：ϕ(n)\phi(n)ϕ(n) 表示小于 nnn 且与 nnn 互素的正整数的个数。
    • 例：ϕ(1)=1\phi(1) = 1ϕ(1)=1。
    • 对于素数 ppp，ϕ(p)=p−1\phi(p) = p - 1ϕ(p)=p−1。
  • 计算公式：
    若 n=p1r1p2r2…pkrkn = p_1^{r_1} p_2^{r_2} \dots p_k^{r_k}n=p1r1​​p2r2​​…pkrk​​，则

    ϕ(n)=n(1−1p1)(1−1p2)…(1−1pk)\phi(n) = n \left(1 - \frac{1}{p_1}\right) \left(1 - \frac{1}{p_2}\right) \dots \left(1 - \frac{1}{p_k}\right) ϕ(n)=n(1−p1​1​)(1−p2​1​)…(1−pk​1​)

2. 欧拉定理 (Euler’s Theorem)

  • 内容：设 a,na, na,n 是互素的正整数（gcd⁡(a,n)=1\gcd(a, n)=1gcd(a,n)=1），则：

    aϕ(n)≡1(modn)a^{\phi(n)} \equiv 1 \pmod n aϕ(n)≡1(modn)

  • 作用：用于降幂运算，或者求模逆（a−1≡aϕ(n)−1(modn)a^{-1} \equiv a^{\phi(n)-1} \pmod na−1≡aϕ(n)−1(modn)）。

3. 费马小定理 (Fermat’s Little Theorem)

这是欧拉定理在模为素数时的特例。

  • 内容：设 ppp 是素数。
    1. 若 p∤ap \nmid ap∤a（即 aaa 不是 ppp 的倍数），则 ap−1≡1(modp)a^{p-1} \equiv 1 \pmod pap−1≡1(modp)。
    2. 对于任意整数 aaa，都有 ap≡a(modp)a^p \equiv a \pmod pap≡a(modp)。

19.6 初等数论在计算机科学技术中的几个应用

1. 产生均匀伪随机数

  • 线性同余法：

    xn=(axn−1+c) mod mx_n = (ax_{n-1} + c) \bmod m xn​=(axn−1​+c)modm

    • x0x_0x0​：种子 (Seed)。
    • aaa：乘数。
    • ccc：增量。
    • mmm：模数。
  • 生成的序列虽然是确定性的，但在统计上具有随机性，称为伪随机数。

2. 密码学基础

  • 凯撒密码 (Caesar Cipher)：简单的移位，E(x)=(x+k) mod 26E(x) = (x + k) \bmod 26E(x)=(x+k)mod26。
  • 仿射密码：E(x)=(ax+b) mod 26E(x) = (ax + b) \bmod 26E(x)=(ax+b)mod26。解密需要求 aaa 的逆。

3. RSA 公钥密码体制

利用大数分解困难性的非对称加密算法。

  • 密钥生成步骤：
    1. 选两个大素数 p,qp, qp,q。
    2. 计算 n=pqn = pqn=pq。
    3. 计算欧拉函数 ϕ(n)=(p−1)(q−1)\phi(n) = (p-1)(q-1)ϕ(n)=(p−1)(q−1)。
    4. 选一个整数 eee（公钥指数），满足 1<e<ϕ(n)1 < e < \phi(n)1<e<ϕ(n) 且 gcd⁡(e,ϕ(n))=1\gcd(e, \phi(n)) = 1gcd(e,ϕ(n))=1。
    5. 计算 ddd（私钥指数），使得 ed≡1(modϕ(n))ed \equiv 1 \pmod{\phi(n)}ed≡1(modϕ(n)) （即 ddd 是 eee 的模逆）。
  • 公钥：(e,n)(e, n)(e,n) —— 公开给所有人。
  • 私钥：(d,n)(d, n)(d,n) —— 自己保密。
  • 加密过程：
    将明文 MMM 转化为整数（M<nM < nM<n），密文 CCC 计算为：

    C=Me mod nC = M^e \bmod n C=Memodn

  • 解密过程：
    收到密文 CCC，恢复明文 MMM：

    M=Cd mod nM = C^d \bmod n M=Cdmodn

4. 模幂算法 (Square and Multiply)

  • 问题：RSA加密解密需要计算很大的 Me mod nM^e \bmod nMemodn，直接乘会溢出且太慢。
  • 快速模幂法：
    将指数 eee 表示为二进制形式，通过反复的平方和乘法并同时取模，将时间复杂度从线性级降低到对数级。
